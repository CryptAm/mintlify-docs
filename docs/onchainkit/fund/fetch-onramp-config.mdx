---
title: fetchOnrampConfig
---


Утилити `fetchOnrampConfig` получает список стран, поддерживаемых Coinbase Onramp, и их доступные способы оплаты. Это полезно, когда вам нужно проверить соответствие пользователя требованиям или отобразить доступные варианты оплаты.

## Использование

<CodeGroup>
```tsx code
import { fetchOnrampConfig } from '@coinbase/onchainkit/fund';

// При использовании с OnchainKitProvider
const config = await fetchOnrampConfig();

// При использовании без OnchainKitProvider или в не-React окружении
const config2 = await fetchOnrampConfig('your-api-key');
```

```js return value
{
  countries: [
    {
      id: "US",
      subdivisions: ["CA", "NY", "TX"],
      paymentMethods: [
        {
          id: "CARD",
          name: "Credit or debit card",
          description: "Pay with Visa or Mastercard",
          currencies: ["USD", "EUR"],
          minAmount: { value: "10.00", currency: "USD" },
          maxAmount: { value: "1000.00", currency: "USD" }
        }
      ]
    }
  ]
}
```
</CodeGroup>

## Параметры

```typescript
/**
 * Опциональный API-ключ для Coinbase Onramp. Если не предоставлен, будет использован
 * API-ключ из OnchainKitProvider. Требуется при использовании утилиты без
 * OnchainKitProvider или в не-React окружении.
 */
apiKey?: string;
```

## Возвращает

`Promise<OnrampConfigResponseData>` - Возвращает промис, который разрешается в данные конфигурации, содержащие поддерживаемые страны и их способы оплаты.

Смотрите [`OnrampConfigResponseData`](/onchainkit/fund/types#onrampconfigresponsedata) для деталей типа ответа.
