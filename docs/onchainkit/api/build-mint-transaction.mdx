Функция `buildMintTransaction` используется для получения неподписанной транзакции для минта NFT.

Перед использованием убедитесь, что у вас есть [Client API Key](https://portal.cdp.coinbase.com/projects/api-keys/client-key) от Coinbase Developer Platform.

{/* ## Использование  */}

{/* <CodeGroup>
```tsx code
import { setOnchainKitConfig } from '@coinbase/onchainkit';
import { buildMintTransaction } from '@coinbase/onchainkit/api';

const response = await buildMintTransaction({
  mintAddress: '0x...',
  takerAddress: '0x...',
  tokenId: '1',
  quantity: 1,
  network: 'networks/base-mainnet',
});
```

```json return value

    "from": "0x...",
```
