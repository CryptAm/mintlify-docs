---
title: getPortfolios
openapi: get /getPortfolios
---


Функция `getPortfolios` возвращает объект, содержащий массив портфолио для 
предоставленных адресов. Каждое портфолио представляет собой объект с адресом кошелька, 
фиатной стоимостью портфолио и массивом токенов, принадлежащих предоставленному адресу.

<Note>
Перед использованием этой конечной точки убедитесь, что у вас есть [Client API Key](https://portal.cdp.coinbase.com/projects/api-keys/client-key)
от Coinbase Developer Platform.
</Note>


<Note>
Обратите внимание: `getPortfolios` доступен только для Base mainnet и Ethereum mainnet. 
Вы можете управлять сетью в `OnchainKitProvider`, устанавливая свойство `chain`.
</Note>
