title: isBase
---


Утилити `isBase` предназначена для проверки, является ли идентификатор цепи (chain id) валидным идентификатором Base или Base Sepolia.

## Использование

<CodeGroup>
```tsx code
import { isBase } from '@coinbase/onchainkit';

// Base Mainnet (chain ID: 8453)
isBase({ chainId: 8453 }); // возвращает true

// Base Sepolia (chain ID: 84532)
isBase({ chainId: 84532 }); // возвращает true

// Ethereum (chain ID: 1)
isBase({ chainId: 1 }); // возвращает false
```

```ts return value
true; // When chainId is 8453 (Base Mainnet) or 84532 (Base Sepolia)
false; // For all other chain IDs
```
</CodeGroup>

## Возвращает

`boolean` - Возвращает `true`, если идентификатор цепи является Base или Base Sepolia, в противном случае `false`.

## Параметры

[`IsBaseOptions`](./types#isbaseoptions) - Смотрите [`IsBaseOptions`](./types#isbaseoptions) для получения подробной информации.
