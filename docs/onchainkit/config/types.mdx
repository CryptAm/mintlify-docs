## `AppConfig`

```ts
type AppConfig = {
  appearance?: {
    name?: string | null; // Название вашего приложения
    logo?: string | null; // URL логотипа вашего приложения
    mode?: Mode | null; // Опционально определяет цветовую схему на основе предпочтений ОС или выбора пользователя
    theme?: ComponentTheme | null; // Опционально устанавливает визуальный стиль для компонентов
  };
  paymaster?: string | null; // URL Paymaster для спонсирования газа
};
```

## `isBaseOptions`

```ts
type isBaseOptions = {

};
```

```ts
type isEthereumOptions = {

```

## `OnchainKitConfig`

```ts
type OnchainKitConfig = {
  address: Address | null; // Адрес опционален, так как у новых пользователей его может не быть
  apiKey: string | null; // ApiKey для API Coinbase Developer Platform
  chain: Chain; // Цепь (chain) должна быть указана, так как нужно знать, какую цепь использовать
  config?: AppConfig; // Опции конфигурации для приложения
  rpcUrl: string | null; // RPC URL для ончейн-запросов. По умолчанию использует CDP Node, если установлен API Key
  schemaId: EASSchemaUid | null; // SchemaId опционален, так как не всем приложениям нужно использовать EAS
  projectId: string | null; // ProjectId из Coinbase Developer Platform, требуется только для поддержки Coinbase Onramp
};
```

## `OnchainKitContextType`

```ts
type OnchainKitContextType = OnchainKitConfig;
```
