Установите и настройте OnchainKit вместе с Next.js.
Если вы интегрируете OnchainKit в существующий проект,
перейдите сразу к разделу [установки OnchainKit](#install-onchainkit).

<Steps>
  <Step title="Установите Next.js">
  Создайте новый проект Next.js с помощью Next.js CLI. 
  Подробную информацию о Next.js можно найти [здесь](https://nextjs.org/docs/getting-started/installation).

  ```bash npm
  npx create-next-app@14
  ```

  В процессе настройки вам будет предложено несколько опций.
  Убедитесь, что вы включили TypeScript, ESLint и Tailwind CSS.
  </Step>
  <Step title="Установите OnchainKit">
  Установите OnchainKit в ваш проект.

  <CodeGroup>
  ```bash npm
  npm install @coinbase/onchainkit
  ```

  ```bash yarn
  yarn add @coinbase/onchainkit
  ```

  ```bash pnpm
  pnpm add @coinbase/onchainkit
  ```

  ```bash bun
  bun add @coinbase/onchainkit
  ```
  </CodeGroup>
  </Step>
  <Step title="Получите ваш Client API Key">
  Получите ваш [Client API Key](https://portal.cdp.coinbase.com/projects/api-keys/client-key) от Coinbase Developer Platform.

<Frame>
  <img
    alt="OnchainKit copy Client API Key"
    src="/images/onchainkit/copy-api-key-guide.png"
    height="364"
  />
</Frame>

  Создайте файл `.env` в корневой директории вашего проекта.

<Frame>
  <img
    alt="OnchainKit define Client API Key"
    src="/images/onchainkit/getting-started-create-env-file.png"
    width="250"
    loading="lazy"
  />
</Frame>

  Добавьте ваш Client API Key в файл `.env`:

  ```tsx .env
  NEXT_PUBLIC_ONCHAINKIT_API_KEY=YOUR_CLIENT_API_KEY;
  ```
  </Step>
  <Step title="Добавьте провайдеры">
  Создайте файл `providers.tsx`. Добавьте `OnchainKitProvider` с нужной вам конфигурацией.

  Внутри OnchainKit автоматически создаст наших рекомендованных провайдеров Wagmi и QueryClient. 
  Если вы хотите их кастомизировать, ознакомьтесь с разделом [Пользовательские 
  дополнительные провайдеры](/onchainkit/config/supplemental-providers).

  ```tsx providers.tsx
  // @noErrors: 2307 2580 2339 - cannot find 'process', cannot find './wagmi', cannot find 'import.meta'
  'use client';

  import type { ReactNode } from 'react';
  import { OnchainKitProvider } from '@coinbase/onchainkit';
  import { base } from 'wagmi/chains'; // добавьте baseSepolia для тестирования // [!code ++]

  export function Providers(props: { children: ReactNode }) {
    return (
      <OnchainKitProvider
        apiKey={process.env.NEXT_PUBLIC_ONCHAINKIT_API_KEY} // [!code ++]
        chain={base} // добавьте baseSepolia для тестирования // [!code ++]
      >
        {props.children}
      </OnchainKitProvider>
    );
  }
  ```
  </Step>
  <Step title="Оберните ваше приложение в <Providers />">
  После настройки оберните ваше приложение компонентом `<Providers />` созданным выше.

  ```javascript app.tsx
  import './globals.css';
  import { Providers } from './providers'; // [!code ++]

  export default function RootLayout({
    children,
  }: Readonly<{
    children: React.ReactNode,
  }>) {
    return (
      <html lang="en">
        <body>
          <Providers> 
            {children}
          </Providers>
        </body>
      </html>
    );
  }
  ```
  </Step>
  <Step title="Добавьте стили">
  Компоненты OnchainKit поставляются с предустановленными стилями. Чтобы подключить эти стили в ваш проект, добавьте следующую строку импорта в начало соответствующего файла:

  ```javascript
  import '@coinbase/onchainkit/styles.css';
  ```

  Например, если вы используете Next.js с маршрутизатором приложения (`app router`), ваш файл `app/layout.tsx` может выглядеть так:

  ```tsx layout.tsx
  import '@coinbase/onchainkit/styles.css'; // [!code ++]
  import './globals.css';
  import type { Metadata } from 'next';
  import { Inter } from 'next/font/google';
  import { headers } from 'next/headers';
  import { type ReactNode } from 'react';
  import { cookieToInitialState } from 'wagmi';

  import { getConfig } from '../wagmi';
  import { Providers } from './providers';

  const inter = Inter({ subsets: ['latin'] });

  export const metadata: Metadata = {
    title: 'Create Wagmi',
    description: 'Generated by create-wagmi',
  };

  export default function RootLayout(props: { children: ReactNode }) {
    const initialState = cookieToInitialState(
      getConfig(),
      headers().get('cookie')
    );
    return (
      <html lang="en">
        <body className={inter.className}>
          <Providers initialState={initialState}>{props.children}</Providers>
        </body>
      </html>
    );
  }
  ```

  Это гарантирует, что стили OnchainKit будут загружены и применены ко всему вашему приложению.

  - Пользователям Tailwind CSS рекомендуем ознакомиться с нашим [Руководством по интеграции с Tailwind](/onchainkit/guides/tailwind).

  - Изменяйте внешний вид компонентов с помощью наших встроенных тем или создавайте свою собственную тему.
    Изучите возможности в нашем [Руководстве по темам](/onchainkit/guides/themes).

  </Step>
</Steps>
