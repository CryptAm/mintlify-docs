---
title: getAvatars
---

Утилита `getAvatars` предназначена для получения нескольких аватаров из провайдера ончейн-идентификации
для массива ENS имен или Basenames в одном пакетном запросе.

Рассмотрите использование утилиты вместо хука, если
вы работаете с Next.js или любым Node.js бэкендом.

## Использование

Получение аватаров для нескольких ENS имен:

<CodeGroup>
```tsx code
import { getAvatars } from '@coinbase/onchainkit/identity';

const ensNames = ['vitalik.eth', 'zizzamia.eth'];
const avatars = await getAvatars({ ensNames });

````

```ts return value
[
  'https://ipfs.io/ipfs/QmSP4nq9fnN9dAiCj42ug9Wa79rqmQerZXZch82VqpiH7U/image.gif',
  'https://ipfs.io/ipfs/QmQ9RT2SrZ6TWUjrQxG4bNnhb3nDqBE5Ld1j9GYvk3kTjf'
]
````

</CodeGroup>

Получение аватаров для нескольких Basenames:

<CodeGroup>
```tsx code
import { getAvatars } from '@coinbase/onchainkit/identity';
import { base } from 'viem/chains';

const ensNames = ['zizzamia.base.eth', 'coinbase.base.eth'];
const avatars = await getAvatars({ ensNames, chain: base });

````

```ts return value
[
  'https://ipfs.io/ipfs/QmQ9RT2SrZ6TWUjrQxG4bNnhb3nDqBE5Ld1j9GYvk3kTjf',
  'https://ipfs.io/ipfs/QmXHFnSXcN7CSuVcyF3vyPgPcPvLQyLpKYBaNpx3x3bPAZ'
]
````

</CodeGroup>

Получение аватаров для комбинации ENS имен и Basenames:

<CodeGroup>
```tsx code
import { getAvatars } from '@coinbase/onchainkit/identity';
import { base } from 'viem/chains';

const ensNames = ['vitalik.eth', 'zizzamia.base.eth'];
const avatars = await getAvatars({ ensNames, chain: base });

````

```ts return value
[
  'https://ipfs.io/ipfs/QmSP4nq9fnN9dAiCj42ug9Wa79rqmQerZXZch82VqpiH7U/image.gif',
  'https://ipfs.io/ipfs/QmQ9RT2SrZ6TWUjrQxG4bNnhb3nDqBE5Ld1j9GYvk3kTjf'
]
````

</CodeGroup>
