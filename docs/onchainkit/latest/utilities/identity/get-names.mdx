---
title: getNames
---

Утилита `getNames` предназначена для получения нескольких имен из провайдера 
ончейн-идентификации для массива адресов в одном пакетном запросе.

Рассмотрите использование утилиты вместо хука, если
вы работаете с Next.js или любым Node.js бэкендом.

## Использование

Получение ENS имен по нескольким адресам:

<CodeGroup>
```tsx code
import { getNames } from '@coinbase/onchainkit/identity';

const addresses = [
'0x4bEf0221d6F7Dd0C969fe46a4e9b339a84F52FDF',
'0x838aD0EAE54F99F1926dA7C3b6bFbF617389B4D9'
];
const names = await getNames({ addresses });

````

```ts return value
['paulcramer.eth', 'vitalik.eth']
````

</CodeGroup>

Получение Basenames по нескольким адресам:

<CodeGroup>
```tsx code
import { getNames } from '@coinbase/onchainkit/identity';
import { base } from 'viem/chains';

const addresses = [
'0x4bEf0221d6F7Dd0C969fe46a4e9b339a84F52FDF',
'0x4bEf0221d6F7Dd0C969fe46a4e9b339a84F52FDF'
];
const names = await getNames({ addresses, chain: base });

````

```ts return value
['paul.base.eth', 'coinbase.base.eth']
````

</CodeGroup>

Получение комбинации ENS имен и усеченных адресов, если у некоторых адресов нет имен:

<CodeGroup>
```tsx code
import { getNames } from '@coinbase/onchainkit/identity';

const addresses = [
'0x4bEf0221d6F7Dd0C969fe46a4e9b339a84F52FDF',
'0x1234567890abcdef1234567890abcdef12345678'
];
const names = await getNames({ addresses });

````

```ts return value
['paulcramer.eth', null]
````

</CodeGroup>

## Возвращает

```typescript
Promise<GetNameReturnType[]>

type GetNameReturnType = string | Basename | null;

type Basename = BaseMainnetName | BaseSepoliaName;
type BaseMainnetName = `${string}.base.eth`;
type BaseSepoliaName = `${string}.basetest.eth`;
```

## Параметры

```typescript
type GetNamesParams = {
  /** Массив адресов Ethereum для получения имен */
  addresses: Address[];
  /** Необязательная цепь для разрешения домена */
  chain?: Chain;
};

type Address = `0x${string}`;

type Chain = {
  id: number;
  name: string;
  // ... другие свойства цепи
};
```
