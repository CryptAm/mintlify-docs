---
title: getName
---

Утилита `getName` предназначена для получения имени из провайдера 
ончейн-идентификации по конкретному адресу.

Рассмотрите использование утилиты вместо хука, если
вы работаете с Next.js или любым Node.js бэкендом.

## Использование

Получение ENS имени по адресу:

<CodeGroup>
```tsx code
import { getName } from '@coinbase/onchainkit/identity';

const address = '0x02feeb0AdE57b6adEEdE5A4EEea6Cf8c21BeB6B1';
const name = await getName({ address });

````

```ts return value
zizzamia.eth;
````

</CodeGroup>

Получение Basename по адресу:

<CodeGroup>
```tsx code
import { getName } from '@coinbase/onchainkit/identity';
import { base } from 'viem/chains';

const address = '0x02feeb0AdE57b6adEEdE5A4EEea6Cf8c21BeB6B1';
const name = await getName({ address, chain: base });

````

```ts return value
zizzamia.base.eth;
````

</CodeGroup>

Получение усеченного адреса, если у адреса нет ENS имени:

<CodeGroup>
```tsx code
import { getName } from '@coinbase/onchainkit/identity';

const address = '0x1234567890abcdef1234567890abcdef12345678';
const name = await getName({ address });

````

```ts return value
0x123...5678
````

</CodeGroup>

## Возвращает

```typescript
Promise<GetNameReturnType>

type GetNameReturnType = string | Basename | null;

type Basename = BaseMainnetName | BaseSepoliaName;
type BaseMainnetName = `${string}.base.eth`;
type BaseSepoliaName = `${string}.basetest.eth`;
```

## Параметры

```typescript
type GetNameParams = {
  /** Адрес Ethereum для разрешения (резолва) */
  address?: Address;
  /** Необязательная цепь для разрешения домена */
  chain?: Chain;
};

type Address = `0x${string}`;

type Chain = {
  id: number;
  name: string;
  // ... другие свойства цепи
};
```
