---
title: buildDepositToMorphoTx
---

Создает массив `Call[]` для внесения активов в Morpho. Передайте вызовы в`<Transaction />` чтобы выполнить депозит.

## Использование

```tsx code
import { buildDepositToMorphoTx } from '@coinbase/onchainkit/earn';

const calls = await buildDepositToMorphoTx({
  vaultAddress: '0x...', // Адрес хранилища Morpho в Base
  tokenAddress: '0x...', // Адрес токена для депозита
  amount: 1000000000000000000n, // Количество токенов для депозита
  recipientAddress: '0x...', // Адрес получателя
});
```

## Возвращает

```typescript
Call[]

type Call = {
  to: Hex;
  data?: Hex;
  value?: bigint;
};

type Hex = `0x${string}`;
```

## Параметры

```typescript
type DepositToMorphoParams = {
  /** Адрес хранилища Morpho */
  vaultAddress: Address;
  /** Адрес токена для депозита */
  tokenAddress: Address;
  /** Количество токенов для депозита */
  amount: bigint;
  /** Адрес, который может вывести внесенные токены */
  recipientAddress: Address;
};

type Address = `0x${string}`;
```
