---
title: buildWithdrawFromMorphoTx
---

Функция `buildWithdrawFromMorphoTx` используется для создания массива `Call[]`, предназначенного для вывода актива из Morpho. Эти вызовы можно передать в компонент `<Transaction />` для отправки транзакции.

## Использование

```tsx code
import { buildWithdrawFromMorphoTx } from '@coinbase/onchainkit/earn';

const calls = await buildWithdrawFromMorphoTx({
  vaultAddress: '0x...', // Адрес хранилища Morpho в Base
  amount: 1000000000000000000n, // Количество токенов для вывода
  recipientAddress: '0x...', // Адрес получателя
});
```

## Возвращает

```typescript
Call[]

type Call = {
  to: Hex;
  data?: Hex;
  value?: bigint;
};

type Hex = `0x${string}`;
```

## Параметры

```typescript
type WithdrawFromMorphoParams = {
  /** Адрес хранилища Morpho */
  vaultAddress: Address;
  /** Количество токенов для вывода */
  amount: bigint;
  /** Адрес, на который будут отправлены выведенные средства */
  recipientAddress: Address;
};

type Address = `0x${string}`;
```
