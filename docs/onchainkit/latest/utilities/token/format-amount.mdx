Утилита `formatAmount` предназначена для консистентного форматирования чисел.

## Использование

<CodeGroup>
```tsx code
import { formatAmount } from '@coinbase/onchainkit/token';

const amount = formatAmount('10000', { minimumFractionDigits: 2 });
```

```ts return value
'10,000.00'; // if in U.S. если локаль - английский (США)

'10.000,00'; // если локаль - европейская страна
```
</CodeGroup>

## Возвращает

`string` - Отформатированное число.

## Параметры

### FormatAmountOptions

```ts
type FormatAmountOptions = {
  locale?: string; // Локаль пользователя (по умолчанию: локаль браузера)
  minimumFractionDigits?: number; // Минимальное количество знаков после запятой для десятичных чисел
  maximumFractionDigits?: number; // Максимальное количество знаков после запятой для десятичных чисел
};
```
