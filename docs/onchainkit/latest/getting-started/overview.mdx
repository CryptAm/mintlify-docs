OnchainKit - это полнофункциональная React-библиотека для создания ончейн-приложений. Создаете ли вы веб-приложение, мини-приложение или гибридное приложение, OnchainKit предоставляет компоненты и утилиты, которые помогут вам быстро и легко разрабатывать.

От подключения кошелька и транзакций до разрешения идентичности и интеграций с DeFi - OnchainKit покрывает все потребности.

## Единое решение для веб-приложений и мини-приложений

OnchainKit помогает создавать традиционные ончейн веб-приложения, мини-приложения и гибридные приложения (приложения, которые работают в клиентах Farcaster, таких как приложение Base, а также в браузере).

Если вы новичок в мини-приложениях, это — веб-приложения с расширенными возможностями, которые получают дополнительную функциональность, такую как автоматическое подключение кошелька, push-уведомления и многое другое. Ознакомьтесь с [документацией Mini Apps](/mini-apps/overview) для получения дополнительной.

## Философия

OnchainKit стремится предоставить комплексный набор инструментов, сочетающий мощные ончейн-функции с удобным для разработчиков дизайном. Он построен с учетом следующих принципов:

- **Эргономичный дизайн:** Инструменты полного цикла, которые делают сложные ончейн-взаимодействия интуитивно понятными.
- **Проверенные практикой паттерны:** Отраслевые лучшие практики, упакованные в готовые к использованию решения.
- **Целевые компоненты:** Предварительно собранные модули для распространенных ончейн-сценариев.
- **Независимость от фреймворка:** Совместимость с любым React-фреймворком.
- **Расширенные возможности от Base:** Глубокая интеграция с функционалом протокола и экосистемой Base.

## Ключевые функции

OnchainKit предоставляет все необходимое для создания современных ончейн-приложений:

- **Компоненты кошелька** - Интерфейс подключения, выбора и управления.
- **Система идентичности** - Разрешение ENS/Basename и отображение профиля.
- **Инструменты для транзакций** - Создание, отправка и отслеживание статуса транзакций.
- **Интеграция с DeFi** - Обмен токенов, фарминг и управление портфелем.
- **Компоненты для коммерции** - Потоки пополнения, оформления заказа и оплаты.
- **Расширенная кастомизация** - Rende

## Кастомизация

Компоненты OnchainKit разработаны с высокой степенью кастомизации. В дополнение к стандартным пропсам, таким как `className` и `children`, многие компоненты поддерживают **render props** для полного контроля над их пользовательским интерфейсом при сохранении всей базовой функциональности.

Render props позволяют реализовать кастомный дизайн для кнопок, полей ввода и других интерактивных элементов, в то время как OnchainKit обрабатывает сложную ончейн-логику:

```tsx
// Пример: Кастомная кнопка подключения кошелька
<ConnectWallet
  render={({ onClick, status, isLoading }) => (
    <button onClick={onClick} className="my-custom-style">
      {status === 'disconnected' ? 'Connect' : 'Connected'}
    </button>
  )}
/>
```

Компоненты с поддержкой render props включают `ConnectWallet`, `TransactionButton`, `SignatureButton`, различные компоненты `Swap` и компоненты `Fund`. Проверяйте документацию отдельных компонентов для получения информации о конкретных интерфейсах render props.
