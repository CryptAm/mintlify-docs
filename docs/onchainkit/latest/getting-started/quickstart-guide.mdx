Создаете новое приложение? Самый простой способ начать - использовать `npx create-onchain`, который автоматически настроит все за вас.

После запуска npx create-onchain и следования подсказкам у вас будет проект на [Next.js](https://nextjs.org/) с установленным OnchainKit, готовый к работе.

## Создание нового проекта

### Поддержка мини-приложения

Если вы создаете мини-приложение или гибридное приложение, вы можете использовать флаг `--mini` для начальной настройки проекта с поддержкой мини-приложений.

```bash Terminal
npx create-onchain --mini
```

### Традиционное ончейн веб-приложение

Если вы не хотите использовать функции мини-приложений, просто опустите флаг `--mini`. Если вы решите добавить функции мини-приложений позже, вы можете преобразовать свой проект в мини-приложение, внеся небольшие изменения в кодовую базу.

```bash Terminal
npx create-onchain
```

## Настройки CLI

CLI задаст вам несколько простых вопросов, чтобы помочь начать работу.

К ним относятся:

- Имя вашего проекта
- Ваш [Coinbase Developer Platform Client API key](https://portal.cdp.coinbase.com/products/onchainkit) (опционально)
- Включить ли телеметрию (опционально)

После этого просто перейдите в папку вашего проекта с помощью `cd` и выполните npm `install` (или используйте предпочитаемый менеджер пакетов), чтобы установить зависимости.

Наконец, запустите `npm run dev`, чтобы запустить проект и начать разработку!

## Что вы получаете

Сгенерированный проект включает:

- **Каркас Next.js** - Новый проект Next.js 15 с использованием App Router
- **Компоненты OnchainKit** - Предварительно настроенный провайдер и компоненты OnchainKit
- **Интеграцию Wagmi & Viem** - Готовые к использованию и настроенные для сети Base
- **TypeScript** - Полная безопасность типов из коробки
