Возникли проблемы? Это руководство охватывает наиболее распространенные проблемы и способы их решения. Не нашли свою проблему? Проверьте наши [GitHub Issues](https://github.com/coinbase/onchainkit/issues) или спросите в [Discord](https://discord.gg/invite/buildonbase).

## Распространенные проблемы

### Настройка окружения

- **Отсутствует Project ID**

  - Ошибка: "Project ID is required for this component"
  - Решение: Добавьте ваш Project ID в OnchainKitProvider:

```tsx
<OnchainKitProvider
  apiKey="YOUR_API_KEY"
  projectId="YOUR_PROJECT_ID" // Получите это на portal.cdp.coinbase.com
  chain={base}
>
```

- **Контракты недоступны**
  - Ошибка: "Contracts are not available" or "Contracts not available for LifecycleStatus"
  - Решения:
    - Убедитесь, что ваш API-ключ OnchainKit установлен правильно
    - Убедитесь, что вы используете API-ключ с [portal.cdp.coinbase.com](https://portal.cdp.coinbase.com)
    - Проверьте, что ваши переменные окружения правильно экспортируются на клиент

### Зависимости

- **Совместимость версий**
  - Проблема: Ошибки компиляции или TypeScript
  - Решение: Убедитесь, что вы используете совместимые версии пакетов:

```json
{
  "dependencies": {
    "@coinbase/onchainkit": "^1.0.0",
    "react": "^19",
    "react-dom": "^19",
    "viem": "^2.27.0",
    "wagmi": "^2.16.0"
  }
}
```

### Конфигурация провайдера

- **Отсутствует OnchainKitProvider**

  - Проблема: Отсутствуют значения контекста OnchainKitProvider
  - Решение: Оберните ваше приложение в OnchainKitProvider. Подробности см. в нашем [setup guide](/getting-started/manual-installation):

```tsx
import { OnchainKitProvider } from '@coinbase/onchainkit';
import { base } from 'viem/chains';

export default function App({ children }) {
  return (
    <OnchainKitProvider
      apiKey={process.env.NEXT_PUBLIC_ONCHAINKIT_API_KEY}
      chain={base}
    >
      {children}
    </OnchainKitProvider>
  );
}
```

### Подключение кошелька

- **Не удалось подключить**

  - Ошибка: "Unable to connect wallet"
  - Решения:
    - Убедитесь, что расширение вашего кошелька установлено и разблокировано
    - Проверьте, что вы находитесь в поддерживаемой сети
    - Попробуйте переключить сеть в вашем кошельке
    - Убедитесь, что ваши RPC-эндпоинты работают

- **Проблемы с переключением сети**
  - Ошибка: "Failed to switch chain"
  - Решения:
    - Убедитесь, что цепь поддерживается OnchainKit
    - Проверьте, что ваш кошелек имеет правильные разрешения
    - Убедитесь, что ваши RPC-эндпоинты настроены корректно
    - Попробуйте сначала вручную добавить цепь в ваш кошелек

### Проблемы с транзакциями

- **Не удалось оценить газ**
  - Ошибка: "Gas estimation failed"
  - Решения:
    - Убедитесь, что у вас достаточно ETH для оплаты комиссий за газ
    - Перепроверьте параметры вашей транзакции
    - Убедитесь, что вы находитесь в правильной сети

### Разрешение модулей

- **Ошибки разрешения модулей**
  - Ошибка: "Cannot find module ... or its corresponding type declarations"
  - Решение: OnchainKit поддерживает только ES Modules, а не CommonJS. Для лучшего опыта используйте Node 22+ и установите `"moduleResolution": "NodeNext"` в вашем конфиге TypeScript.
