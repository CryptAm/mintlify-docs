---
title: useBuildDepositToMorphoTx
---


Хук `useBuildDepositToMorphoTx` используется для создания транзакции депозита в Morpho из React-компонента.

Хук обрабатывает следующее:

- Получение данных, необходимых для создания транзакции
- Использование [`buildDepositToMorphoTx`](/onchainkit/api/build-deposit-to-morpho-tx) для создания транзакции
- Возврат вызовов транзакции для использования с компонентом `<Transaction />`

## Использование

```tsx code
import { useBuildDepositToMorphoTx } from '@coinbase/onchainkit/earn';

const { calls } = useBuildDepositToMorphoTx({
  vaultAddress: '0x...',
  recipientAddress: '0x...',
  amount: '1000000000000000000',
});
```

## Возвращает

```ts
{
  calls: Call[];
}

type Call = { 
  to: Hex; 
  data?: Hex; 
  value?: bigint; 
};
```

## Параметры

```ts
type UseBuildDepositToMorphoTxParams = {
  vaultAddress: Address;
  recipientAddress?: Address;
  amount: string;
};
```
