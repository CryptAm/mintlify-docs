---
title: useBuildWithdrawFromMorphoTx
---


Хук `useBuildWithdrawFromMorphoTx` используется для создания транзакции вывода из Morpho из React-компонента.

Хук обрабатывает следующее:

- Получение данных, необходимых для создания транзакции
- Использование [`buildWithdrawFromMorphoTx`](/onchainkit/api/build-withdraw-from-morpho-tx) для создания транзакции
- Возврат вызовов транзакции для использования с компонентом `<Transaction />`

## Использование

```tsx code
import { useBuildWithdrawFromMorphoTx } from '@coinbase/onchainkit/earn';

const { calls } = useBuildWithdrawFromMorphoTx({
  vaultAddress: '0x...',
  recipientAddress: '0x...',
  amount: '1000000000000000000',
  tokenDecimals: 18,
});
```

## Возвращает

```ts
{
  calls: Call[];
}

type Call = { 
  to: Hex; 
  data?: Hex; 
  value?: bigint; 
};
```

## Параметры

```ts
type UseBuildWithdrawFromMorphoTxParams = {
  vaultAddress: Address;
  recipientAddress?: Address;
  amount: string;
  tokenDecimals: number | undefined;
};
```
