---
title: useAddress
---

Хук `useAddress` используется для получения адреса из провайдера ончейн-идентификации по заданному имени.

Он реализован с использованием [useQuery](https://tanstack.com/query/latest/docs/framework/react/reference/useQuery) из `@tanstack/react-query`, и возвращает объект `UseQueryResult`, что позволяет передавать все опции `@tanstack/react-query`.

## Использование

```tsx
import { useAddress } from '@coinbase/onchainkit/identity';

const { data: address, isLoading } = useAddress({ name: 'zizzamia.base.eth' });
```

## Возвращает

```ts
useQuery<Promise<GetAddressReturnType>>

type GetAddressReturnType = Address | null;
```

## Параметры

### UseAddressParams

```ts
type UseAddressParams = {
  /** ENS или Basename, для которого нужно получить адрес Ethereum */
  name: string | Basename;
  /** Необязательная цепь для разрешения домена */
  chain?: Chain;
};
```

### UseQueryOptions

```ts
type UseQueryOptions<TData = unknown> = Omit<
  TanstackUseQueryOptions<TData>,
  'queryKey' | 'queryFn'
>;
```
