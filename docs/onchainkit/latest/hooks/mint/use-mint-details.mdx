---
title: useMintDetails
---


Хук `useMintDetails` реализует API `getMintDetails`, возвращая данные, необходимые для просмотра NFT перед его минтом.

Он реализован с использованием [useQuery](https://tanstack.com/query/latest/docs/framework/react/reference/useQuery) из `@tanstack/react-query`, и возвращает объект `UseQueryResult`, что позволяет передавать все опции `@tanstack/react-query`.

Перед использованием убедитесь, что вы получили [Client API Key](https://portal.cdp.coinbase.com/projects/api-keys/client-key) из Coinbase Developer Platform.

## Использование

<CodeGroup>
```tsx code
import { useMintDetails } from '@coinbase/onchainkit/nft';

const Component = () => {
  const { data, isLoading, error } = useMintDetails({
    contractAddress: '0x...',
    takerAddress: '0x...',
    tokenId: '1',
  });

  if (isLoading) return <div>Loading...</div>;
  
  return <div>{JSON.stringify(data)}</div>;
};
```

## Возвращает

```ts
UseQueryResult<MintDetails>

type MintDetails = {
  /** Название NFT */
  name: string;
  /** Описание NFT */
  description: string;
  /** URL изображения NFT */
  imageUrl: string;
  /** URL анимации NFT */
  animationUrl: string;
  /** MIME-тип NFT */
  mimeType: string;
  /** ERC721, ERC1155 */
  contractType: ContractType;
  /** Цена NFT */
  price: NFTPrice;
  /** Комиссия за минт NFT */
  mintFee: NFTPrice;
  /** Максимальное количество минтов на один кошелек */
  maxMintsPerWallet: number;
  /** Имеет ли пользователь право на минт */
  isEligibleToMint: boolean;
  /** Адрес создателя NFT */
  creatorAddress: Address;
  /** Общее количество токенов */
  totalTokens: string;
  /** Общее количество владельцев NFT */
  totalOwners: string;
  /** Сеть, в которой находится NFT */
  network: string;
};
```

## Параметры

```ts
type GetMintDetailsParams = {
  /** Адрес контракта токена */
  contractAddress: Address;
  /** Адрес пользователя */
  takerAddress?: Address;
  /** ID токена (обязательно для ERC1155) */
  tokenId?: string;
};
```
