---
title: useTokenDetails
---


Хук `useTokenDetails` реализует API `getTokenDetails`, 
возвращая данные, необходимые для просмотра NFT.

Он реализован с использованием [useQuery](https://tanstack.com/query/latest/docs/framework/react/reference/useQuery) из `@tanstack/react-query`, и возвращает объект `UseQueryResult`, что позволяет передавать все опции `@tanstack/react-query`.

Перед использованием убедитесь, что вы получили [Client API Key](https://portal.cdp.coinbase.com/projects/api-keys/client-key) из Coinbase Developer Platform.

## Использование

<CodeGroup>
```tsx code
import { useTokenDetails } from '@coinbase/onchainkit/nft';

const Component = () => {
  const { data, isLoading, error } = useTokenDetails({
    contractAddress: '0x...',
    tokenId: '1',
  });

  if (isLoading) return <div>Loading...</div>;
  
  return <div>{JSON.stringify(data)}</div>;
};
```

```tsx scaffolding
import { QueryClient, QueryClientProvider } from '@tanstack/react-query';
import { OnchainKitProvider } from '@coinbase/onchainkit';
import { useTokenDetails } from '@coinbase/onchainkit/nft';

const queryClient = new QueryClient();

<QueryClientProvider client={queryClient}>
  <OnchainKitProvider
    apiKey={...apiKey}
    chain={base}
  >
    <Component />
  </OnchainKitProvider>
</QueryClientProvider>
```

## Возвращает

```ts
UseQueryResult<TokenDetails>

type TokenDetails = {
  /** Название токена */
  name: string;
  /** Описание токена */
  description: string;
  /** URL изображения токена */
  imageUrl: string;
  /** URL анимации токена */
  animationUrl: string;
  /** MIME-тип токена  */
  mimeType: string;
  /** Адрес владельца токена */
  ownerAddress: Address;
  /** Цена последней продажи токена */
  lastSoldPrice: NFTPrice;
  /** ERC721, ERC1155 */
  contractType: ContractType;
};
```

## Параметры

```ts
type GetTokenDetailsParams = {
  /** Адрес контракта токена */
  contractAddress: Address;
  /** ID токена */
  tokenId?: string;
};
```
