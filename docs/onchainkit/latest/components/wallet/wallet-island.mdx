`WalletIsland` - это наша флагманская реализация новых расширенных компонентов
кошелька, предоставляющая продвинутый интерфейс кошелька для пользователей, включая:

- QR-код для получения средств
- ссылку для покупки криптовалюты за фиат
- интерфейс обмена
- токеновый портфель пользователя
- кнопку, которая перемещается в любую точку экрана

Разработанный для настольных устройств, компонент `WalletIsland` предоставляет
пользователям плавный способ взаимодействия со своим кошельком и управления активами.

<Note>
Для работы этого компонента требуется установить `projectId` и `apiKey` в `OnchainKitProvider`. Вы можете найти оба этих значения на [Coinbase Developer Platform](https://portal.cdp.coinbase.com/products/onchainkit).
</Note>

<Note>
Пожалуйста, обратите внимание: балансы токенов доступны только для Base mainnet и Ethereum mainnet. 
Вы можете управлять сетью в `OnchainKitProvider`, установив свойство `chain`.
</Note>

## Быстрый старт

Расширенный `Wallet` имеет две реализации по умолчанию:

- `WalletAdvancedDefault`
- `WalletIsland`

`WalletAdvancedDefault` предоставляет пользователю расширенный интерфейс 
кошелька, привязанный к экрану, как и наш стандартный компонент `Wallet`.
`WalletIsland` предоставляет такой же мощный интерфейс, одновременно позволяя
пользователю перетаскивать компонент по окну.

Если вам нужна большая кастомизация, следуйте руководству по реализации ниже.

```tsx
import { WalletAdvancedDefault } from '@coinbase/onchainkit/wallet';
import { WalletIsland } from '@coinbase/onchainkit/wallet';
<>
  <WalletAdvancedDefault />
  <WalletIsland />
</>
```

Вы увидите две кнопки кошелька ниже. Прямо ниже находится `WalletAdvancedDefault`.
Эта версия останется закрепленной в текущей позиции.

`WalletIsland` находится в правом нижнем углу окна. Не стесняйтесь перетаскивать
его.

Хотя это поведение более заметно с компонентом `WalletIsland`, вы
заметите, что для обоих компонентов контейнер `WalletAdvanced` будет появляться в
разных позициях в зависимости от расположения компонента в окне:

- Если справа достаточно места, он будет выровнен по левому краю.
- Если справа недостаточно места, он будет выровнен по правому краю.
- Если ниже достаточно места, он откроется ниже.
- Если ниже недостаточно места, он откроется выше.

<Note>
Если вы пытаетесь использовать этот компонент, а кнопка покупки не работает, 
убедитесь, что у вас правильно настроен `projectId` в ваших провайдерах.

Если история транзакций не загружается, убедитесь, что у вас правильно настроен
`apiKey` в ваших провайдерах.

</Note>

## Настройка `WalletIsland` и расширенного `Wallet`

Испытайте магию, просто добавив компоненты `<Wallet />` и расширенный `Wallet`,
и наблюдайте, как ваш компонент плавно оживает.

Как и [`WalletDefault`](/onchainkit/wallet/wallet) , `WalletAdvancedDefault` использует
несколько компонентов[`<Identity>`](/onchainkit/identity/identity), такие как [`<Avatar>`](/onchainkit/identity/avatar),
[`<Name>`](/onchainkit/identity/name), и [`<Address>`](/onchainkit/identity/address).

Мы представили несколько расширенных компонентов кошелька, включая
`<WalletAdvancedAddressDetails>`, `<WalletAdvancedTokenHoldings>`, и
`<WalletAdvancedTransactionActions>`.

```tsx
import {
  ConnectWallet,
  Wallet,
  WalletDropdown,
  WalletAdvancedAddressDetails,
  WalletAdvancedTokenHoldings,
  WalletAdvancedTransactionActions,
  WalletAdvancedWalletActions,
} from '@coinbase/onchainkit/wallet';
import { Address, Avatar, Name, Identity } from '@coinbase/onchainkit/identity';
import { color } from '@coinbase/onchainkit/theme';

export function YourWalletAdvanced() {
  return (
    <div className="flex justify-end">
      <Wallet>
        <ConnectWallet>
          <Avatar className="h-6 w-6" />
          <Name />
        </ConnectWallet>
        <WalletDropdown>
          <WalletAdvancedWalletActions />
          <WalletAdvancedAddressDetails />
          <WalletAdvancedTransactionActions />
          <WalletAdvancedTokenHoldings />
        </WalletDropdown>
      </Wallet>
    </div>
  );
}
```

При кастомизации вашей реализации расширенного `Wallet` используйте свойство `draggable` на
`Wallet`, чтобы включить возможность перетаскивания. `draggable` по умолчанию `false`, но
когда `draggable` установлен в `true`, вы также можете установить свойство `draggableStartingPosition`, чтобы
указать начальную позицию вашего `WalletIsland`.

```tsx
import {
  ConnectWallet,
  Wallet,
  WalletDropdown,
  WalletAdvancedAddressDetails,
  WalletAdvancedTokenHoldings,
  WalletAdvancedTransactionActions,
  WalletAdvancedWalletActions,
} from '@coinbase/onchainkit/wallet';
import { Address, Avatar, Name, Identity } from '@coinbase/onchainkit/identity';
import { color } from '@coinbase/onchainkit/theme';

export function DraggableWalletAdvanced() {
  return (
    <div className="flex justify-end">
      <Wallet
        draggable={true} // [!code focus]
        draggableStartingPosition={{ // [!code focus]
          x: window.innerWidth - 300, // [!code focus]
          y: window.innerHeight - 100, // [!code focus]
        }}
      >
        <ConnectWallet>
          <Avatar className="h-6 w-6" />
          <Name />
        </ConnectWallet>
        <WalletDropdown>
          <WalletAdvancedWalletActions />
          <WalletAdvancedAddressDetails />
          <WalletAdvancedTransactionActions />
          <WalletAdvancedTokenHoldings />
        </WalletDropdown>
      </Wallet>
    </div>
  );
}
```

## Компоненты

Компоненты предназначены для совместной работы иерархически. Для каждого компонента убедитесь в следующем:

- `<Wallet />` - Служит основным контейнером для всех компонентов, связанных с кошельком.
- `<WalletDropdown />` - Содержит дополнительную информацию о кошельке и опции. Поместите внутрь компонента `<Wallet />`.
- `<WalletAdvancedWalletActions />` - Предоставляет действия с кошельком, такие как просмотр истории транзакций, просмотр QR-кода и отключение кошелька. Поместите внутрь компонента `<WalletDropdown />`.
- `<WalletAdvancedAddressDetails />` - Отображает адрес пользователя, аватар и баланс портфеля в фиате. Поместите внутрь компонента `<WalletDropdown />`.
- `<WalletAdvancedTransactionActions />` - Кнопки для покупки криптовалюты за фиат, перевода криптовалюты и обмена. Поместите внутрь компонента `<WalletDropdown />`.
- `<WalletAdvancedTokenHoldings />` - Отображает балансы токенов и их стоимость в фиате. Поместите внутрь компонента `<WalletDropdown />`.
- `<ConnectWallet />` - Обрабатывает процесс подключения кошелька. Поместите дочерние компоненты внутрь, чтобы настроить внешний вид кнопки подключения.
- `<AppWithWalletModal />` - Включает агрегированный опыт кошелька.
- `<Identity />` - Отображает информацию о личности пользователя. Поместите внутрь `<WalletDropdown />` для полного просмотра профиля.
- `<WalletDropdownBasename />` - Отображает Basename пользователя в выпадающем списке.
- `<WalletDropdownLink />` - Создает пользовательскую ссылку в выпадающем списке. Используйте свойство `icon` для добавления иконки и `href` для указания назначения.
- `<WalletDropdownDisconnect />` - Предоставляет опцию отключения в выпадающем списке.

Дополнительные компоненты для кастомизации интерфейса кошелька включают:

- `<Avatar />` - Отображает аватар пользователя.
- `<Name />` - Показывает имя пользователя или ENS.
- `<Badge />` - Может использоваться для отображения дополнительного статуса или информации о пользователе.
- `<Address />` - Показывает адрес кошелька пользователя.
- `<EthBalance />` - Отображает баланс ETH пользователя.

Компонент Wallet автоматически обрабатывает состояние подключения кошелька и соответствующим образом обновляет UI.
Вам необходимо обернуть ваше приложение или соответствующую часть его этими компонентами,
чтобы обеспечить полный опыт взаимодействия с кошельком.
