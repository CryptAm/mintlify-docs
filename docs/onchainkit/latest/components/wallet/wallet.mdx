Компоненты `Wallet` предоставляют интерфейс для пользователей, чтобы подключить их смарт-аккаунт с их идентификационной информацией, такой как Basename и баланс ETH.

Они имеют встроенные отполированные пользовательские интерфейсы как для веба, так и для мобильного веба, что делает невероятно простым их улучшение с помощью готовых компонентов.

## Быстрый старт

Чтобы оптимизировать процесс интеграции и интегрировать упрощенную версию компонента `Wallet`, вы можете отобразить компонент `Wallet` без дочерних элементов.

Если вам нужна большая кастомизация, следуйте руководству по реализации для нашего компонента `Wallet` ниже, где вы можете вручную определить каждый подкомпонент и проп.

```tsx
import { Wallet } from '@coinbase/onchainkit/wallet';

<Wallet />
```

  ```tsx
      }),

      [baseSepolia.id]: http(),

  });

  function App({ children }: { children: ReactNode }) {

  }
  ```

## Кастомизация текста и стиля кнопки подключения

Каждый компонент OnchainKit предлагает гибкость для кастомизации `className`
и настройки стиля React компонентов, которые он представляет.

Настройте текст подключения кошелька, используя свойство `disconnectedLabel` на компоненте `<ConnectWallet />`.

### Расширенная кастомизация с свойствами рендера

Для полного контроля над отображением кнопки ConnectWallet используйте свойство `render`. Это позволяет реализовать пользовательский UI, сохраняя всю функциональность кошелька:

```tsx
import { ConnectWallet } from '@coinbase/onchainkit/wallet';

<ConnectWallet
  render={({ label, onClick, status, isLoading }) => (
    <button 
      onClick={onClick}
      className="my-custom-button-class"
      disabled={isLoading}
    >
      {status === 'disconnected' ? 'Connect Your Wallet' : label}
    </button>
  )}
/>
```


## Пример использования

### Использование с Sign In With Ethereum (SIWE)

Чтобы использовать [Sign In With Ethereum (SIWE)](https://docs.login.xyz/general-information/siwe-overview) с OnchainKit, вы можете использовать свойство `onConnect` в компоненте `<ConnectWallet />`. Это запустит запрос SIWE, когда пользователь подключит свой кошелек.

```tsx
import { ConnectWallet } from '@coinbase/onchainkit/wallet';
import { base } from 'wagmi/chains';
import { createSiweMessage } from 'viem/siwe'
import { useSignMessage } from 'wagmi';

const message = createSiweMessage({
  address: '0xA0Cf798816D4b9b9866b5330EEa46a18382f251e',
  chainId: base.id,
  domain: 'example.com',
  nonce: 'foobarbaz',
  uri: 'https://example.com/path',
  version: '1',
})

export function WalletComponents() {
  const { signMessage } = useSignMessage();

  return (
    <ConnectWallet onConnect={() => {signMessage({ message })}} />
  );
}

```

## Компоненты

Компоненты предназначены для совместной работы иерархически. Для каждого компонента убедитесь в следующем:

- <Wallet /> - Служит основным контейнером для всех компонентов, связанных с кошельком.
- <ConnectWallet /> - Обрабатывает процесс подключения кошелька. Поместите дочерние компоненты внутрь, чтобы настроить внешний вид кнопки подключения.
- <AppWithWalletModal /> - Включает агрегированный опыт кошелька.
- <WalletDropdown /> - Содержит дополнительную информацию о кошельке и опции. Поместите внутрь компонента <Wallet />.
- <Identity /> - Отображает информацию о личности пользователя. Поместите внутрь <WalletDropdown /> для полного просмотра профиля.
- <WalletDropdownBasename /> - Отображает Basename пользователя в выпадающем списке.
- <WalletDropdownLink /> - Создает пользовательскую ссылку в выпадающем списке. Используйте проп icon для добавления иконки и href для указания назначения.
- <WalletDropdownDisconnect /> - Предоставляет опцию отключения в выпадающем списке.

Дополнительные компоненты для кастомизации интерфейса кошелька включают:

- <Avatar /> - Отображает аватар пользователя.
- <Name /> - Показывает имя пользователя или ENS.
- <Badge /> - Может использоваться для отображения дополнительного статуса или информации о пользователе.
- <Address /> - Показывает адрес кошелька пользователя.
- <EthBalance /> - Отображает баланс ETH пользователя.

Компонент Wallet автоматически обрабатывает состояние подключения кошелька и соответствующим образом обновляет UI.
Вам необходимо обернуть ваше приложение или соответствующую часть его этими компонентами,
чтобы обеспечить полный опыт взаимодействия с кошельком.

## Свойства компонентов

### WalletProps

```ts
type WalletProps = {
  children?: React.ReactNode;
  isSponsored?: boolean; // Спонсировать ли транзакции для функции отправки расширенной реализации кошелька
  className?: string;
} & (
  | { draggable?: true; draggableStartingPosition?: { x: number; y: number } }
  | { draggable?: false; draggableStartingPosition?: never }
);
```

### ConnectWalletProps

```ts
type ConnectWalletProps = {
  children?: ReactNode; // Children могут быть использованы для отображения кастомизированного контента, когда кошелек подключен
  className?: string; // Опциональное переопределение className для элемента button
  onConnect?: () => void; // Опциональная функция обратного вызова для выполнения при подключении кошелька
  disconnectedLabel?: ReactNode; // Опциональное переопределение отображения в отключенном состоянии
  render?: ({
    label,
    onClick,
    context,
    status,
    isLoading,
  }: {
    label: ReactNode;
    onClick: () => void;
    context: WalletContextType;
    status: 'disconnected' | 'connecting' | 'connected';
    isLoading: boolean;
  }) => ReactNode; // Пользовательская функция рендера для полного контроля над отображением кнопки
};
```

### WalletDropdownProps

```ts
type WalletDropdownProps = {
  children?: React.ReactNode;
  className?: string; // Опциональное переопределение className для верхнего элемента div
  classNames?: {
    container?: string;
    qr?: WalletAdvancedQrReceiveProps['classNames'];
    swap?: WalletAdvancedSwapProps['classNames'];
  };
  swappableTokens?: Token[];
};
```

### WalletDropdownBasenameProps

```ts
type WalletDropdownBasenameProps = {
  className?: string; // Опциональное переопределение className для элемента
};
```

### WalletDropdownDisconnectProps

```ts
type WalletDropdownDisconnectProps = {
  className?: string; // Опциональное переопределение className для элемента
  text?: string; // Опциональное переопределение текста для кнопки
};
```

### WalletDropdownLinkProps

```ts
type WalletDropdownLinkProps = {
  children: string;
  className?: string; // Опциональное переопределение className для элемента
  href: string;
  icon?: 'wallet' | ReactNode;
  rel?: string;
  target?: string;
};
```
