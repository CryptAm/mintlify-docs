---
title: "Avatar"
---

Компонент `Avatar` используется для отображения аватара ENS или [Basenames](https://www.base.org/names) связанного с адресами Ethereum. 
Когда аватар недоступен, по умолчанию отображается аватар синего цвета.

## Использование

Адрес с аватаром ENS:
```tsx
import { Avatar } from '@coinbase/onchainkit/identity';
<Avatar address="0x838aD0EAE54F99F1926dA7C3b6bFbF617389B4D9" /> // [!code focus]
```


```tsx

  address="0x1234567890abcdef1234567890abcdef12345678"
/> // [!code focus]
```

Используйте свойство `defaultComponent`, чтобы изменить аватар по умолчанию, когда аватар ENS не найден.
Используйте свойство `loadingComponent`, чтобы изменить плейсхолдер загрузки:
```tsx
import { Avatar } from '@coinbase/onchainkit/identity';
<Avatar
  address="0x1234567890abcdef1234567890abcdef12345678"
  loadingComponent={(
    <div className="h-8 w-8">
      <svg width="100%" height="100%" viewBox="0 0 20 20" xmlns="http://www.w3.org/2000/svg">
        <polygon points="6,1 14,1 19,6 19,14 14,19 6,19 1,14 1,6" fill="yellow" stroke="yellow" stroke-width="1"/>
      </svg>
    </div>
  )}
  defaultComponent={(
    <div className="h-8 w-8">
      <svg width="100%" height="100%" viewBox="0 0 20 20" xmlns="http://www.w3.org/2000/svg">
        <polygon points="6,1 14,1 19,6 19,14 14,19 6,19 1,14 1,6" fill="green" stroke="green" stroke-width="1"/>
      </svg>
    </div>
  )}
/>
```

Показывайте аттестацию на аватаре ENS с помощью компонента [`Badge`](/onchainkit/identity/badge).
Используйте [`OnchainKitProvider`](/onchainkit/config/onchainkit-provider) или компонент [`Identity`](/onchainkit/identity/identity) со свойством `schemaId`.
```tsx
import { Avatar, Badge, Identity } from '@coinbase/onchainkit/identity';
<Identity
  schemaId="0xf8b05c79f090979bf4a80270aba232dff11a10d9ca55c4f88de95317970f0de9"
  address="0x838aD0EAE54F99F1926dA7C3b6bFbF617389B4D9"
>
  <Avatar >
    <Badge />
  </Avatar>
</Identity>
```
