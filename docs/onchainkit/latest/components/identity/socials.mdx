---
title: "Socials"
---

Компонент `Socials` отображает ссылки на социальные сети, связанные с Basenames и ENS-именами.
Он автоматически получает и отображает социальные ссылки, хранящиеся в текстовых записях.

## Возможности
- **Разрешение имен:** Разрешает как Basenames, так и ENS-имена для получения связанных социальных ссылок
- **Поддержка нескольких платформ:** Поддерживает Twitter, GitHub, Discord и другие основные платформы
- **Настраиваемое отображение:** Гибкие опции стилизации для соответствия дизайну вашего приложения
- **Осведомленность о цепи:** Работает с различными EVM-цепями, которые поддерживают ENS

## Свойства

Компонент `Socials` принимает следующие свойства:

| Свойство | Тип  | Описание    |
|----------|------|-------------|
| `address` | `Address \| null` | Адрес Ethereum, для которого нужно разрешить социальные ссылки |
| `ensName` | `string` | Опциональное ENS-имя для разрешения социальных ссылок |
| `chain` | `Chain` | Цепь для использования при разрешении ENS |
| `className` | `string` | Пользовательские CSS-классы для применения к компоненту |

## Обработка ошибок

Компонент обрабатывает различные граничные случаи:

- Возвращает `null`, если социальные ссылки не найдены
- Показывает пустое состояние во время загрузки
- Аккуратно обрабатывает ошибки разрешения ENS
- Проверяет социальные ссылки перед отображением
