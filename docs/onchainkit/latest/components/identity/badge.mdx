Используйте компонент `Badge` вместе с компонентами [`Avatar`](/onchainkit/identity/avatar) или [`Name`](/onchainkit/identity/name), чтобы отображать аттестации пользователя, прикрепленные к его аккаунту.

## Использование

Бейдж с цветами по умолчанию:

<CodeGroup>
```tsx tsx
import { Badge } from '@coinbase/onchainkit/identity';
<Badge className="badge" /> // [!code focus]
```

```css css
.badge {
  background: #4F46E5;
  path {
    fill: #F9FAFB;
  }
}
```

## Бейдж с подсказкой (Tooltip)

Вы можете включить подсказку для бейджа аттестации, чтобы предоставить контекст о том, что он представляет:

```tsx
import { Badge } from '@coinbase/onchainkit/identity';

<Badge 
  tooltip={true} // [!code focus]
  className="badge"
/>
```

## Свойства

[`BadgeReact`](/onchainkit/identity/types#badgereact)

| Свойство | Тип  | Описание    |
|----------|------|-------------|
| `className` | `string` | Опциональное переопределение className для верхнего span элемента |
| `tooltip` | `boolean \| string` | Управляет показом подсказки при наведении на бейдж. При `true` отображает название аттестации. При передаче строки показывает этот кастомный текст. По умолчанию `false` |

## Бейдж на `<Name />` и `<Avatar />`

Бейдж с [`<Name />`](/onchainkit/identity/name), лучше всего использовать, когда [`<Name />`](/onchainkit/identity/name) отображается вместе с компонентами [`<Avatar />`](/onchainkit/identity/avatar).

В обоих примерах мы используем ID схемы Coinbase [Verified Account](https://base.easscan.org/schema/view/0xf8b05c79f090979bf4a80270aba232dff11a10d9ca55c4f88de95317970f0de9), чтобы показать бейдж верификации Coinbase на компонентах Name и Avatar.

```tsx
import { base } from 'viem/chains';
import { Badge, Name, Identity } from '@coinbase/onchainkit/identity';

const address = '0x838aD0EAE54F99F1926dA7C3b6bFbF617389B4D9';
const COINBASE_VERIFIED_ACCOUNT_SCHEMA_ID =
  '0xf8b05c79f090979bf4a80270aba232dff11a10d9ca55c4f88de95317970f0de9';

<Identity
  className="bg-transparent"
  schemaId={COINBASE_VERIFIED_ACCOUNT_SCHEMA_ID}
  address={address}
>
  <Name address={address}>
    <Badge tooltip={true} /> {/* С подсказкой, показывающей название аттестации */}
  </Name>
</Identity>
```

Бейдж с [`<Avatar />`](/onchainkit/identity/avatar), лучше всего использовать, когда [`<Avatar />`](/onchainkit/identity/avatar) не сопровождается никакими подписями.

```tsx
import { base } from 'viem/chains';
import { Avatar, Badge, Identity } from '@coinbase/onchainkit/identity';

const address = '0x838aD0EAE54F99F1926dA7C3b6bFbF617389B4D9';
const COINBASE_VERIFIED_ACCOUNT_SCHEMA_ID =
  '0xf8b05c79f090979bf4a80270aba232dff11a10d9ca55c4f88de95317970f0de9';

<Identity
  className="bg-transparent"
  schemaId={COINBASE_VERIFIED_ACCOUNT_SCHEMA_ID}
>
  <Avatar address={address}>
    <Badge tooltip="Verified by Coinbase" /> {/* With custom tooltip text */}
  </Avatar>
</Identity>
```
