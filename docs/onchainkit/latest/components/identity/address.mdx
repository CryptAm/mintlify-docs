title: "Address"

Компонент `Address` используется для отображения сокращенного адреса аккаунта пользователя.

## Использование

Усеченный адрес аккаунта:

```tsx
import { Address } from '@coinbase/onchainkit/identity';
<Address address="0x02feeb0AdE57b6adEEdE5A4EEea6Cf8c21BeB6B1" /> // [!code focus]
```

### Отображение полного адреса

Установите `isSliced` в значение false, чтобы отобразить полный адрес:

```tsx
import { Address } from '@coinbase/onchainkit/identity';
<Address address="0x02feeb0AdE57b6adEEdE5A4EEea6Cf8c21BeB6B1" isSliced={false} /> // [!code focus]
```

### Переопределение стилей

Вы можете переопределить стили компонента, используя `className`.

```tsx
import { Address } from '@coinbase/onchainkit/identity';
<Address
  className="bg-emerald-400 px-2 py-1 rounded" // [!code focus]
  address="0x02feeb0AdE57b6adEEdE5A4EEea6Cf8c21BeB6B1"
/>
```

## Свойства

[`AddressProps`](#addressprops)

```ts
type AddressProps = {
  /** Адрес Ethereum для отображения. */
  address?: Address | null;
  /** Опциональное переопределение className для верхнего span элемента. */
  className?: string;
  /** Определяет, должен ли отображаемый адрес быть усеченным. (по умолчанию: true) */
  isSliced?: boolean;
  /** По умолчанию true. Опциональный boolean для отключения функции копирования адреса при клике. */
  hasCopyAddressOnClick?: boolean;
};
```
