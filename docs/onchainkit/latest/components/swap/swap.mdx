Компоненты `Swap` предоставляют комплексный интерфейс для пользователей для выполнения обменов [Token](/onchainkit/token/types#token).

## Быстрый старт

Компонент `Swap` можно использовать двумя способами: как полностью готовый компонент или как настраиваемый контейнер. Когда вы предоставляете свойства to и from, он отображает полный интерфейс обмена, включая `SwapAmountInput`, `SwapSettings`, `SwapToggleButton`, `SwapButton`, и `SwapToast`.

Для большей кастомизации используйте подход с `children`, показанный в следующем примере.

```tsx
import { Swap } from '@coinbase/onchainkit/swap'; // [!code focus]
import type { Token } from '@coinbase/onchainkit/token';

const eth: Token = {
  name: 'ETH',
  address: '',
  symbol: 'ETH',
  decimals: 18,
  image:
    'https://wallet-api-production.s3.amazonaws.com/uploads/tokens/eth_288.png',
  chainId: 8453,
};

const usdc: Token = {
  name: 'USDC',
  address: '0x833589fcd6edb6e08f4c7c32d4f71b54bda02913',
  symbol: 'USDC',
  decimals: 6,
  image:
    'https://d3r81g40ycuhqg.cloudfront.net/wallet/wais/44/2b/442b80bd16af0c0d9b22e03a16753823fe826e5bfd457292b55fa0ba8c1ba213-ZWUzYjJmZGUtMDYxNy00NDcyLTg0NjQtMWI4OGEwYjBiODE2',
  chainId: 8453,
};

<Swap // [!code focus]
  from={[eth]} // [!code focus]
  to={[usdc]} // [!code focus]
/> // [!code focus]
```

### Поддерживаемые роутеры обмена

Компонент `Swap` поддерживает два роутера обмена:

- [Uniswap V3](https://app.uniswap.org/) (по умолчанию)
- [0x Aggregator](https://0x.org/)

Чтобы использовать 0x Aggregator, установите свойство `experimental.useAggregator` в `true`.

### Спонсирование газа с помощью Paymaster

Чтобы спонсировать транзакции обмена для ваших пользователей, переключите Paymaster с помощью свойства `isSponsored`.

По умолчанию будет использоваться Paymaster [Coinbase Developer Platform](https://portal.cdp.coinbase.com/products/bundler-and-paymaster).

Вы можете настроить параметры спонсирования на странице [Paymaster](https://portal.cdp.coinbase.com/products/bundler-and-paymaster).
По соображениям безопасности мы рекомендуем настроить список разрешенных контрактов в портале. Без определенного списка разрешенных контрактов ваш Paymaster сможет спонсировать только до $1.

Контракт, используемый в нашем Swap API, - это [Universal Router](https://basescan.org/address/0x3fC91A3afd70395Cd496C647d5a6CC9D4B2b7FAD) от Uniswap,, который развернут на Base по адресу `0x3fC91A3afd70395Cd496C647d5a6CC9D4B2b7FAD`.

Обратите внимание, что спонсирование газа будет работать только для смарт-аккаунтов.

## Компоненты

Компоненты предназначены для совместной работы иерархически. Для каждого компонента убедитесь в следующем:

- `<Swap />` - Установите адрес пользователя и обработку ошибок.
- `<SwapAmountInput />` - Установите [Token](/onchainkit/token/types#token) для обмена и укажите тип ввода (`from` или `to`).
- `<SwapToggleButton />` - Опциональный компонент для переключения между типами ввода.
- `<SwapMessage />` - Опциональный компонент, который отображает сообщение, связанное с текущим состоянием операции обмена.
- `<SwapButton />` - Установите колбэки onSuccess и onError.
- `<SwapToast />` - Опциональный компонент для уведомления пользователя об успешной транзакции обмена.

## Props

### SwapProps

```ts
type SwapProps = {
  className?: string; // Опциональное переопределение className для верхнего элемента div
  config?: {
    maxSlippage: number; //Максимально допустимое проскальзывание для обмена (например, 3 для 3%). Это процент, а не базисные пункты
  };
  experimental?: {
    useAggregator: boolean; // Использовать ли агрегатор DEX. true - 0x Aggregator, false - Uniswap V3 (по умолчанию: false)
  };
  isSponsored?: boolean; // Опциональная настройка для спонсирования обменов с помощью Paymaster. (по умолчанию: false)
  onError?: (error: SwapError) => void; // Опциональная функция обратного вызова для обработки ошибок внутри провайдера
  onStatus?: (lifecycleStatus: LifecycleStatus) => void; // Опциональная функция обратного вызова, которая предоставляет доступ к состоянию жизненного цикла компонента
  onSuccess?: (transactionReceipt: TransactionReceipt) => void; // Опциональная функция обратного вызова, которая предоставляет доступ к квитанции транзакции
} & (
  | {
      children: ReactNode;
      to?: never;
      from?: never;
      disabled?: never;
      title?: never;
      headerLeftContent?: never;
    }
  | {
      children?: never;
      to?: Token[];
      from?: Token[];
      disabled?: boolean;
      title?: ReactNode; // Заголовок для компонента Swap. (по умолчанию: "Обмен")
      headerLeftContent?: ReactNode; // Левая часть заголовка для компонента Swap (например, кнопка назад)
    }
);
```

  | { statusName: 'slippageChange'; statusData: LifecycleStatusDataShared }
  | { statusName: 'transactionPending'; statusData: LifecycleStatusDataShared }
  | { statusName: 'transactionApproved'; statusData: { transactionHash?: Hex; transactionType: SwapTransactionType } & LifecycleStatusDataShared }
