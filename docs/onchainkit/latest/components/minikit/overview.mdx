MiniKit - это официальный SDK для создания мини-приложений, которые бесшовно работают в Base App и других клиентах Farcaster. Он предоставляет React хуки, управление контекстом и интеграцию кошельков, что устраняет сложности создания социально-нативных приложений.

## Архитектура

MiniKit состоит из трех основных компонентов:

### `<OnchainKitProvider miniKit={{ enabled: true }}>`
Провайдер контекста React, который настраивает:
- Настройку Wagmi и react-query
- Конфигурацию цепи и API-ключа
- Настройки темы и внешнего вида
- Интеграцию коннектора Farcaster

### Хуки
React хуки для взаимодействия с фреймами:
- **Управление фреймами**: `useMiniKit`, `useAddFrame`, `useClose`
- **Навигация**: `useOpenUrl`, `useViewProfile`, `useViewCast`
- **Социальные функции**: `useComposeCast`, `useNotification`
- **Аутентификация**: `useAuthenticate`
- **UI**: `usePrimaryButton`

### CLI-инструменты
Утилиты командной строки для:
- Создания проекта с помощью `npx create-onchain --mini`
- Генерации манифеста с помощью `npx create-onchain --manifest`
- Автоматизации рабочего процесса разработки

## Ключевые концепции

### Жизненный цикл фрейма
Мини-приложения работают внутри фреймов Farcaster и должны сигнализировать о готовности:
- **Инициализируйте** MiniKitProvider в корне вашего приложения
- **Сигнализируйте о готовности** с помощью setFrameReady() в вашем основном компоненте
- **Обрабатывайте взаимодействия** через хуки MiniKit

### Контекст vs Аутентификация
- **Данные контекста** (`useMiniKit().context`) предоставляют информацию о пользователе, но могут быть подделаны
- **Аутентификация ** (`useAuthenticate()`) предоставляет криптографически проверенную идентификацию пользователя
- Всегда используйте аутентификацию для операций, критичных к безопасности

### Кросс-клиентная совместимость
MiniKit автоматически обнаруживает клиентскую среду и адаптируется:
- **Base App**: Нативная интеграция кошелька, расширенные функции
- **Farcaster**: Соответствие стандартному протоколу Farcaster
- **Другие клиенты**: Аккуратные фоллбэки и режимы совместимости
