---
title: useOpenUrl
description: Открытие URL-адресов внутри или вне контекста фрейма
---

Определен в [@coinbase/onchainkit](https://github.com/coinbase/onchainkit)

<Info>
Открывает URL-адреса в соответствующем контексте - либо внутри среды мини-приложения, либо во внешних браузерах. Автоматически обрабатывает кросс-клиентную совместимость.
</Info>

## Возвращает

<ResponseField name="openUrl" type="(url: string) => void">
Функция, которая открывает указанный URL-адрес. Поведение зависит от типа URL и возможностей клиента.

<ParamField body="url" type="string" required>
URL-адрес для открытия. Могут быть http/https ссылки, диплинки или другие URI схемы.
</ParamField>
</ResponseField>

<RequestExample>
```tsx components/ExternalLinks.tsx
import { useOpenUrl } from '@coinbase/onchainkit/minikit';

export default function ExternalLinks() {
  const openUrl = useOpenUrl();

  return (
    <div className="external-links">
      <button onClick={() => openUrl('https://base.org')}>
        Visit Base.org
      </button>
      <button onClick={() => openUrl('https://twitter.com/base')}>
        Follow on Twitter
      </button>
      <button onClick={() => openUrl('https://discord.gg/basechain')}>
        Join Discord
      </button>
    </div>
  );
}
```

```tsx components/SmartLink.tsx
import { useOpenUrl } from '@coinbase/onchainkit/minikit';

export default function SmartLink({ href, children, className }) {
  const openUrl = useOpenUrl();

  const handleClick = (e) => {
    e.preventDefault();
    openUrl(href);
  };

  return (
    <a 
      href={href} 
      onClick={handleClick}
      className={className}
    >
      {children} ↗
    </a>
  );
}
```
</RequestExample>

<Warning>
URL-адреса, открытые с помощью `useOpenUrl`, могут открываться в разных контекстах в зависимости от клиента. Не полагайтесь на конкретное поведение открытия для критически важной функциональности.
</Warning>

<Info>
`useOpenUrl` автоматически обрабатывает кросс-клиентную совместимость и обеспечивает лучший пользовательский опыт для внешней навигации.
</Info>
