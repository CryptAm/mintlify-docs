title: useViewProfile
description: Навигация к профилям Farcaster из вашего мини-приложения
---

Определен в [@coinbase/onchainkit](https://github.com/coinbase/onchainkit)

<Info>
Открывает профили пользователей Farcaster в хост-приложении. По умолчанию использует профиль текущего пользователя, если FID не указан.
</Info>

## Параметры

<ParamField body="fid" type="string" optional>
Farcaster ID профиля для просмотра. Если не указан, по умолчанию используется FID текущего пользователя из контекста MiniKit.
</ParamField>

## Возвращает

<ResponseField name="viewProfile" type="() => void">
Функция, которая открывает указанный профиль Farcaster при вызове.
</ResponseField>

<RequestExample>
```tsx components/ProfileButton.tsx
import { useViewProfile } from '@coinbase/onchainkit/minikit';

export default function ProfileButton() {
  const viewProfile = useViewProfile(); // Использует FID текущего пользователя

  return (
    <button onClick={viewProfile}>
      View My Profile
    </button>
  );
}
```

```tsx components/Leaderboard.tsx
export default function Leaderboard({ players }) {

      <span className="score">{player.score}</span>
      <button 
        onClick={viewProfile}
        className="profile-btn"
      >
        View Profile
      </button>
    </div>
  );
}
```

```tsx components/SocialActions.tsx
import { useViewProfile, useMiniKit } from '@coinbase/onchainkit/minikit';

export default function SocialActions() {
  const { context } = useMiniKit();
  const viewMyProfile = useViewProfile(); // Текущий пользователь
  const viewHostProfile = useViewProfile(context.client.clientFid); // Профиль хост-приложения

  return (
    <div className="social-actions">
      <button onClick={viewMyProfile}>
        My Profile
      </button>
      
      <button onClick={viewHostProfile}>

      </button>
    </div>
  );
}
```

## Паттерны использования

### Обнаружение пользователей
Позволяйте пользователям исследовать профили других участников:

```tsx components/ProfileList.tsx
const ProfileList = ({ users }) => {
  return (
    <div className="user-list">
      {users.map(user => (
        <UserProfileCard 
          key={user.fid}
          fid={user.fid}
          name={user.name}
        />
      ))}
    </div>
  );
};

const UserProfileCard = ({ fid, name }) => {
  const viewProfile = useViewProfile(fid);
  
  return (
    <div onClick={viewProfile} className="profile-card">
      <h4>{name}</h4>
      <span>FID: {fid}</span>
    </div>
  );
};
```

### Социальные игры
Соединяйте игроков в многопользовательских играх:

```tsx components/GameLobby.tsx
const GameLobby = ({ players }) => {
  return (
    <div className="game-lobby">
      <h3>Players in Game</h3>
      {players.map(player => (
        <PlayerChip key={player.fid} player={player} />
      ))}
    </div>
  );
};

const PlayerChip = ({ player }) => {
  const viewProfile = useViewProfile(player.fid);
  
  return (
    <div className="player-chip" onClick={viewProfile}>
      {player.username}
    </div>
  );
};
```

### Атрибуция создателей
Ссылайтесь на создателей контента и соавторов:

```tsx components/ContentAttribution.tsx
const ContentAttribution = ({ creator }) => {

```

## Рекомендации

### Пользовательский опыт
- **Четкий призыв к действию**: Используйте описательный текст кнопки, например "Посмотреть профиль" или имена пользователей
- **Визуальная обратная связь**: Обозначайте кликабельные элементы профиля соответствующим стилем
- **Осведомленность о контексте**: Показывайте соответствующие действия профиля на основе отношений пользователя

### Оптимизация производительности
- **Мемоизация обработчиков профиля**: Используйте тот же экземпляр хука для того же FID
- **Пакетная загрузка данных профиля**: Загружайте информацию профиля эффективно при отображении нескольких пользователей

```tsx components/ProfileActions.tsx
import { useMemo } from 'react';

const ProfileActions = ({ userFid }) => {
  const viewProfile = useViewProfile(userFid);
  
  // Мемоизация, чтобы избежать повторного создания обработчика
  const handleProfileView = useMemo(() => viewProfile, [viewProfile]);
  
  return (
    <button onClick={handleProfileView}>
      View Profile
    </button>
  );
};
```

### Доступность
- **Навигация с клавиатуры**: Обеспечьте доступность ссылок на профиль с клавиатуры
- **Поддержка скринридеров**: Используйте семантический HTML и ARIA-метки
- **Управление фокусом**: Корректно обрабатывайте фокус при возврате из просмотра профилей

```tsx components/AccessibleProfileLink.tsx
const AccessibleProfileLink = ({ fid, userName }) => {
  const viewProfile = useViewProfile(fid);
  
  return (
    <button 
      onClick={viewProfile}
      aria-label={`View ${userName}'s Farcaster profile`}
    >
      {userName}
    </button>
  );
};
```

<Info>
Поведение просмотра профиля может отличаться между клиентами Farcaster. В Base App профили открываются в контексте приложения. В других клиентах опыт может отличаться в зависимости от их реализации.
</Info>

<Warning>
Всегда проверяйте FID перед передачей в `useViewProfile`. Неверные FID могут вызвать ошибки или неожиданное поведение в хост-приложении.
</Warning>
