Компонент `TokenImage` - это изображение, которое обрезает изображение токена до круга с регулируемым размером.

Если у свойства `token` нет изображения, отображается часть символа токена и детерминированный темный цвет.

## Использование

`TokenImage` with an url

```tsx
import { TokenImage } from '@coinbase/onchainkit/token';
<div>
// ---cut-before---
<TokenImage
  token={{
    image: "https://dynamic-assets.coinbase.com/dbb4b4983bde81309ddab83eb598358eb44375b930b94687ebe38bc22e52c3b2125258ffb8477a5ef22e33d6bd72e32a506c391caa13af64c00e46613c3e5806/asset_icons/4113b082d21cc5fab17fc8f2d19fb996165bcce635e6900f7fc2d57c4ef33ae9.png",
  }}
  size={24}
/>

<TokenImage
  token={{
```

## Свойства

### TokenImageProps

```ts
type TokenImageProps = {
  className?: string; // Опциональный дополнительный CSS-класс для применения к компоненту
  size?: number; // размер изображения в px (по умолчанию: 24)
  token: Token;
};
```
