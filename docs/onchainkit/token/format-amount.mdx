---
title: formatAmount
---


Утилита `formatAmount` предназначена для консистентного форматирования чисел.

## Использование

<CodeGroup>
```tsx code
import { formatAmount } from '@coinbase/onchainkit/token';

const amount = formatAmount('10000', { minimumFractionDigits: 2 });
```

```ts return value
'10,000.00'; // если локаль U.S. English

'10.000,00'; // если локаль страны EU
```
</CodeGroup>

## Возвращает

`string` - Отформатированное количество.

## Параметры

[`FormatAmountOptions`](/onchainkit/token/types#formatamountoptions)
