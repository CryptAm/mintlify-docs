## `FormatAmountOptions`

```ts
type FormatAmountOptions = {
  locale?: string; // Локаль пользователя (по умолчанию: локаль браузера)
  minimumFractionDigits?: number; // Минимальное количество цифр дробной части числа (по умолчанию: 0)
  maximumFractionDigits?: number; // Максимальное количество цифр дробной части числа (по умолчанию: 0)
};
```

## `FormatAmountResponse`

```ts
type FormatAmountResponse = string; // См. Number.prototype.toLocaleString для получения дополнительной информации
```

## `Token`

```ts
type Token = {
  address: Address; // Адрес контракта токена
  chainId: number; // Идентификатор цепи (chain id) контракта токена
  decimals: number; // Количество десятичных знаков токена
  image: string | null; // URL-строка логотипа токена
  name: string;
  symbol: string; // Тикер-символ или сокращение, до 11 символов
};
```

## `TokenChipReact`

```ts
type TokenChipReact = {
  token: Token; // Отображаемый токен
  onClick?: (token: Token) => void;
  className?: string;
  isPressable?: boolean; // По умолчанию: true
};
```

## `TokenImageReact`

```ts
type TokenImageReact = {
  className?: string; // Необязательный дополнительный CSS-класс для применения к компоненту
  size?: number; // размер изображения в пикселях (px) (по умолчанию: 24)
  token: Token;
};
```
