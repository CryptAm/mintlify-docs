## `BuyReact`

```ts
type BuyReact = {
  className?: string; // Опциональное переопределение класса для верхнего div-элемента.
  config?: {
    maxSlippage: number; // Максимально допустимое проскальзывание (slippage) для обмена (например, 3 для 3%). Это процент, а не базисные пункты.
  };
  experimental?: {
    /**
     * Использовать ли агрегатор DEX.
     * true - Агрегатор 0x
     * false - Uniswap V3
     * @default false
     */
    useAggregator: boolean;
  };
  isSponsored?: boolean; // Опциональная настройка для спонсирования обменов с помощью Paymaster. (по умолчанию: false)
  onError?: (error: SwapError) => void; // Опциональная функция обратного вызова для обработки ошибок в провайдере.
  onStatus?: (lifecycleStatus: LifecycleStatus) => void; // Опциональная функция обратного вызова, предоставляющая состояние жизненного цикла компонента.
  onSuccess?: (transactionReceipt?: TransactionReceipt) => void; // Опциональная функция обратного вызова, предоставляющая квитанцию о транзакции.
  fromToken?: Token; // Опциональный токен, с которого производится обмен. (По умолчанию поддерживаются USDC и ETH)
  toToken: Token; // Токен для покупки.
};
```
