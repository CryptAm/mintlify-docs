Компоненты `Buy` предоставляют комплексный интерфейс для пользователей, чтобы покупать [Токены](/onchainkit/token/types#token).

Компонент `Buy` по умолчанию поддерживает обмен токенов из USDC и ETH с возможностью предоставить дополнительный токен по выбору с помощью свойства `fromToken`. Кроме того, пользователи могут покупать токены, используя свой аккаунт Coinbase, Apple Pay или дебетовую карту.

<Note>
Опции Apple Pay и дебетовой карты доступны только для активов, поддерживаемых Coinbase.
</Note>


Перед использованием убедитесь, что вы выполнили все шаги из ['Начало работы'](/onchainkit/getting-started).

<Note>
Для этого компонента требуется установить `projectId` в `OnchainKitProvider`. Вы можете найти свой `projectId` в [Coinbase Developer Platform](https://portal.cdp.coinbase.com/products/onchainkit).
</Note>

```tsx
import type { Token } from '@coinbase/onchainkit/token';

export default function BuyComponents() { // [!code focus]
  const degenToken: Token = {

    symbol: 'DEGEN',

    <Buy toToken={degenToken} /> // [!code focus]
  ) // [!code focus]
} // [!code focus]

```

### Спонсирование газа через Paymaster

Чтобы спонсировать транзакции обмена для ваших пользователей, включите Paymaster с помощью свойства `isSponsored`.

По умолчанию будет использоваться Paymaster от [Coinbase Developer Platform](https://portal.cdp.coinbase.com/products/bundler-and-paymaster).

Вы можете настроить параметры спонсирования на странице [Paymaster](https://portal.cdp.coinbase.com/products/bundler-and-paymaster) page.
По соображениям безопасности мы рекомендуем настроить список разрешенных контрактов в Порталe. Без определенного списка разрешенных контрактов ваш Paymaster сможет спонсировать только до $1.

Контракт, используемый в нашем Swap API, - это [Universal Router](https://basescan.org/address/0x3fC91A3afd70395Cd496C647d5a6CC9D4B2b7FAD) от Uniswap, который развернут на Base по адресу `0x3fC91A3afd70395Cd496C647d5a6CC9D4B2b7FAD`.

Обратите внимание, что спонсирование газа будет работать только для Smart Wallets.

```tsx
import { Buy } from '@coinbase/onchainkit/buy'; // [!code focus]
import type { Token } from '@coinbase/onchainkit/token';

export default function BuyComponents() { // [!code focus]
  const degenToken: Token = {
    name: 'DEGEN',
    address: '0x4ed4e862860bed51a9570b96d89af5e1b0efefed',
    symbol: 'DEGEN',
    decimals: 18,
    image:
    'https://d3r81g40ycuhqg.cloudfront.net/wallet/wais/3b/bf/3bbf118b5e6dc2f9e7fc607a6e7526647b4ba8f0bea87125f971446d57b296d2-MDNmNjY0MmEtNGFiZi00N2I0LWIwMTItMDUyMzg2ZDZhMWNm',
    chainId: 8453,
  };

  return ( // [!code focus]
    <Buy toToken={degenToken} isSponsored /> // [!code focus]
  ) // [!code focus]
} // [!code focus]
```
