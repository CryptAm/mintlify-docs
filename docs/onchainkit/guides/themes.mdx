## Обзор

OnchainKit предоставляет гибкий контроль над внешним видом через две основные функции: `mode` и `theme`.

- **Режим**: Управляет светлой/темной темой и включает опцию `auto`, которая наследует системные настройки.
- **Тема**: Определяет общее стилистическое оформление всех компонентов.

Вы можете выбирать из встроенных тем или динамически переключать режимы в зависимости от предпочтений пользователя или системных настроек, что позволяет создать кастомизированный и адаптивный пользовательский интерфейс.

## Встроенные темы

OnchainKit предлагает несколько тем для быстрого оформления ваших компонентов. Установите тему через `OnchainKitProvider`, используя `config.appearance.theme`:

- `default`: Включает как светлый, так и темный режим.
- `base`: Только один режим.
- `cyberpunk`: Только один режим.
- `hacker`: Только один режим.
- `custom`: Только один режим.

Если тема не выбрана, автоматически применяется тема `default`.

```tsx
// @noErrors:  2304 17008 1005
<OnchainKitProvider
  apiKey={process.env.NEXT_PUBLIC_ONCHAINKIT_API_KEY}
  chain={base}
  config={{ // [!code focus]
    appearance: { // [!code focus]
      mode: 'auto', // 'auto' | 'light' | 'dark'
      theme: 'default', // 'default' | 'base' | 'cyberpunk' | 'hacker' // [!code focus]
    }, // [!code focus]
  }} // [!code focus]
>
```

## Режим

Управляйте цветовой схемой, устанавливая свойство `config.appearance.mode` у `OnchainKitProvider`:

- `auto`: Автоматически переключается между светлой и темной темой на основе предпочтений операционной системы пользователя.
- `light`: Принудительно использует светлую версию темы для всех компонентов.
- `dark`: Принудительно использует темную версию темы для всех компонентов.

Если режим не указан, по умолчанию будет применен режим `auto`.

```tsx
// @noErrors:  2304 17008 1005
<OnchainKitProvider
  apiKey={process.env.NEXT_PUBLIC_ONCHAINKIT_API_KEY}
  chain={base}
  config={{
    appearance: {
      mode: 'auto', // 'auto' | 'light' | 'dark' // [!code focus]
      theme: 'default', // 'default' | 'base' | 'cyberpunk' | 'hacker' | 'your-custom-theme'
    },
  }}
>
```

## CSS Переопределения

Точечно настройте конкретные аспекты существующей темы.
Это полезно, когда вы хотите внести корректировки во внешний вид компонентов без создания полностью новой темы.

```css
@layer base {
  :root
  .default-light,
  .default-dark,
  .base,
  .cyberpunk,
  .hacker {
    /* Переопределите конкретные переменные по мере необходимости */
    --ock-font-family: 'your-custom-value';
    --ock-border-radius: 'your-custom-value';
    --ock-text-primary: 'your-custom-value';
  }
}
```

## Пользовательская тема

Определите полностью новый внешний вид для вашего приложения.
Это дает вам полный контроль над всеми аспектами дизайна, включая цвета, шрифты и другие визуальные свойства.

#### Варианты использования:

##### Автоматическое переключение между светлой и темной темой:

- Чтобы автоматически переключаться между светлой и темной версиями вашей пользовательской темы:

```tsx
// @noErrors:  2304 17008 1005
<OnchainKitProvider
  apiKey={process.env.NEXT_PUBLIC_ONCHAINKIT_API_KEY}
  chain={base}
  config={{
    appearance: {
      mode: 'auto', // [!code focus]
      theme: 'custom', // [!code focus]
    },
  }}
>
```

##### Одна версия темы:

- Чтобы использовать только одну версию вашей пользовательской темы постоянно:

```tsx
// @noErrors:  2304 17008 1005
<OnchainKitProvider
  apiKey={process.env.NEXT_PUBLIC_ONCHAINKIT_API_KEY}
  chain={base}
  config={{
    appearance: {
      mode: 'light', // [!code focus]
      theme: 'custom', // [!code focus]
    },
  }}
>
```
