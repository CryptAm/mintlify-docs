## Инструменты AI

### Replit

[Replit](https://replit.com) - это облачная платформа для программирования, которая упрощает процессы настройки, сборки, публикации и развертывания проектов. Она позволяет разработчикам программировать в среде, похожей на Google Docs, а предустановленные шаблоны предоставляют отличную отправную точку для создания сайта, приложения или игры. Ее новый AI Agent может помочь в процессе разработки кода и работать с несколькими файлами одновременно, делая процесс программирования похожим на личную беседу.

### Cursor

[Cursor](https://cursor.com) - это редактор кода с поддержкой AI, который делает опыт программирования волшебным. Построенный как форк VS Code, он обладает мощными функциями, такими как AI-автодополнение кода, редактирование на естественном языке и понимание кодовой базы. Cursor Pro бесплатен в течение первых двух недель после регистрации и предлагает более мощные модели.

### Использование OnchainKit с CDP SDK

Вы можете использовать OnchainKit вместе с [CDP SDK](https://docs.cdp.coinbase.com/get-started/docs/overview) для доступа к дополнительным возможностям, таким как [AgentKit](https://docs.cdp.coinbase.com/agentkit/docs/welcome).
