Компоненты Swap предоставляют комплексный интерфейс для выполнения пользователями обменов [Token](/onchainkit/token/types#token).

Перед их использованием убедитесь, что вы выполнили все [Getting Started steps](/onchainkit/getting-started).

## Быстрый старт

Компонент `SwapDefault` - это упрощенная версия компонента `Swap`, предназначенная для ускорения процесса интеграции разработчиками. Вместо ручного определения каждого дочернего компонента и свойства, разработчики могут использовать эту сокращенную версию, которая отображает предлагаемую нами реализацию компонента и включает `SwapAmountInput`, `SwapSettings`, `SwapToggleButton`, `SwapButton`, и `SwapToast`.

Если вам требуется больше кастомизации, следуйте руководству по реализации для нашего компонента `Swap` ниже.

```tsx
import { SwapDefault } from '@coinbase/onchainkit/swap'; // [!code focus]
import type { Token } from '@coinbase/onchainkit/token';

const eth: Token = {
  name: 'ETH',
  address: '',
  symbol: 'ETH',
  decimals: 18,
  image:
    'https://wallet-api-production.s3.amazonaws.com/uploads/tokens/eth_288.png',
  chainId: 8453,
};

const usdc: Token = {
  name: 'USDC',
  address: '0x833589fcd6edb6e08f4c7c32d4f71b54bda02913',
  symbol: 'USDC',
  decimals: 6,
  image:
    'https://d3r81g40ycuhqg.cloudfront.net/wallet/wais/44/2b/442b80bd16af0c0d9b22e03a16753823fe826e5bfd457292b55fa0ba8c1ba213-ZWUzYjJmZGUtMDYxNy00NDcyLTg0NjQtMWI4OGEwYjBiODE2',
  chainId: 8453,
};

<SwapDefault // [!code focus]
  from={[eth]} // [!code focus]
  to={[usdc]} // [!code focus]
/> // [!code focus]
```

<Danger>
**Примечание: Этот интерфейс предназначен только для демонстрационных целей.**

Обмен будет выполняться и работать «из коробки», когда вы внедрите компонент в свое собственное приложение.
</Danger>


### Поддерживаемые маршрутизаторы обмена

Компонент `Swap` поддерживает два маршрутизатора обмена:

- [Uniswap V3](https://app.uniswap.org/) (по умолчанию)
- [0x Aggregator](https://0x.org/)

Чтобы использовать 0x Aggregator, установите свойство `experimental.useAggregator` в значение `true`.

### Спонсирование газа через Paymaster

Чтобы спонсировать транзакции обмена для ваших пользователей, активируйте Paymaster с помощью свойства `isSponsored`.

По умолчанию для этого будет использоваться Paymaster от [Coinbase Developer Platform](https://portal.cdp.coinbase.com/products/bundler-and-paymaster).

Вы можете настроить параметры спонсирования на странице [Paymaster](https://portal.cdp.coinbase.com/products/bundler-and-paymaster).
Из соображений безопасности мы рекомендуем настроить разрешенный список контрактов (allowlist) в портале. Без определенного разрешенного списка контрактов ваш Paymaster сможет спонсировать только до $1.

Контракт, используемый в нашем Swap API - это [Universal Router](https://basescan.org/address/0x3fC91A3afd70395Cd496C647d5a6CC9D4B2b7FAD) от Uniswap, который развернут на Base по адресу `0x3fC91A3afd70395Cd496C647d5a6CC9D4B2b7FAD`.

Обратите внимание, что спонсирование газа будет работать только для смарт-кошельков.

## Компоненты

Компоненты предназначены для совместной иерархической работы. Для каждого компонента убедитесь в следующем:

- `<Swap />` - Устанавливает адрес пользователя и обработку ошибок.
- `<SwapAmountInput />` - Устанавливает [Token](/onchainkit/token/types#token) для обмена и указывает тип ввода (`from` или `to`).
- `<SwapToggleButton />` - Опциональный компонент для переключения между типами ввода.
- `<SwapMessage />` - Опциональный компонент, отображающий сообщение, связанное с текущим состоянием операции обмена.
- `<SwapButton />` - Устанавливает колбэки `onSuccess` и `onError`.
- `<SwapToast />` - Опциональный компонент для уведомления пользователя об успешной транзакции обмена.
