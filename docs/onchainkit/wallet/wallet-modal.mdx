`WalletModal` предлагает пользователям несколько вариантов подключения кошелька. Модальное окно автоматически появляется, когда пользователи нажимают на компонент `ConnectWallet`, и предоставляет несколько путей подключения:

1. Быстрое создание нового смарт-кошелька для новых пользователей
2. Подключение Coinbase Wallet (поддерживает как смарт-кошельки, так и EOA)
3. Подключение MetaMask
4. Подключение Phantom
5. Подключения Rabby, Frame и Trust Wallet (опционально)

## Компоненты

Компоненты модального окна кошелька предназначены для совместной иерархической работы:

- `<OnchainKitProvider />` - Настраивает параметры и внешний вид модального окна.
- `<ConnectWallet />` - Запускает открытие модального окна при клике.
- `<WalletModal />` - Сам интерфейс модального окна, содержащий:
  - Процесс создания смарт-кошелька.
  - Подключение Coinbase Wallet.
  - Подключение MetaMask.
  - Подключение Phantom.
  - Подключения Rabby, Trust и Frame Wallet (если включены).
  - Ссылки на условия использования и политику конфиденциальности.

Модальное окно автоматически обрабатывает:

- Состояния подключения кошелька.
- Обработку ошибок.
- Адаптивное поведение для мобильных устройств и компьютеров.
- Настройку темы.
- Отображение условий использования и политики конфиденциальности.

## Поддержка дополнительных кошельков

По умолчанию модальное окно кошелька включает Coinbase Wallet, MetaMask и Phantom в качестве вариантов подключения. Вы можете включить поддержку дополнительных кошельков, настроив объект `supportedWallets` в `OnchainKitProvider`:

```tsx
<OnchainKitProvider
  apiKey={process.env.ONCHAINKIT_API_KEY}
  chain={base}
  config={{
    appearance: {
      name: 'Your App Name',
      logo: 'https://your-logo.com',
      mode: '400',
      theme: 'default',
    },
    wallet: {
      display: 'modal',
      termsUrl: 'https://...',
      privacyUrl: 'https://...',
      supportedWallets: { // [!code focus]
        rabby: true, // [!code focus]
        trust: true, // [!code focus]
        frame: true, // [!code focus]
      }, // [!code focus]
    },
  }}
>
  {children}
</OnchainKitProvider>
```

Каждый кошелек может быть индивидуально включен или отключен по мере необходимости. Если конфигурация `supportedWallets` не предоставлена, дополнительные кошельки по умолчанию будут отключены.
