`WalletIsland` - это наша флагманская реализация новых расширенных компонентов
Wallet, предоставляющая пользователям расширенный интерфейс кошелька, включающий:

- QR-код для получения средств
- Ссылку для покупки криптовалюты за фиат
- Интерфейс обмена (swap)
- Портфель токенов пользователя
- Кнопку, которую можно перемещать в любое место на экране

Разработанный для настольных устройств, компонент `WalletIsland` предоставляет
пользователям плавный способ взаимодействия со своим кошельком и управления активами.

Перед использованием этих компонентов убедитесь, что вы выполнили все [Getting Started steps](/onchainkit/getting-started).

<Note>
Для работы этого компонента требуется, чтобы в `OnchainKitProvider` были установлены `projectId` и `apiKey`. 
Вы можете найти оба этих значения на [Coinbase Developer Platform](https://portal.cdp.coinbase.com/products/onchainkit).
</Note>


<Note>
Обратите внимание: балансы токенов доступны только для основной сети Base и основной сети Ethereum. 
Вы можете управлять сетью в `OnchainKitProvider`, установив свойство `chain`.
</Note>


## Быстрый старт

Расширенный `Wallet` имеет две стандартные реализации:

- `WalletAdvancedDefault`
- `WalletIsland`

`WalletAdvancedDefault` предоставляет пользователю расширенный интерфейс
кошелька, прикрепленный к экрану, как и наш стандартный компонент `Wallet`.
`WalletIsland` предоставляет такой же мощный интерфейс, позволяя 
пользователю перетаскивать компонент по окну.

Если вам требуется больше кастомизации, следуйте руководству по реализации ниже.

```tsx
import { WalletAdvancedDefault } from '@coinbase/onchainkit/wallet';
import { WalletIsland } from '@coinbase/onchainkit/wallet';
<>
  <WalletAdvancedDefault />
  <WalletIsland />
</>
```

Вы увидите две кнопки Wallet ниже. Прямо под этим текстом находится `WalletAdvancedDefault`.
Эта версия останется закрепленной на своей текущей позиции.

`WalletIsland` находится в правом нижнем углу окна. Не стесняйтесь перетаскивать
его.

Хотя это поведение более заметно у компонента `WalletIsland`, вы заметите, что для
обоих компонентов контейнер `WalletAdvanced` будет появляться в разных позициях 
в зависимости от расположения компонента в окне:

- Если справа достаточно места, он будет выровнен по левому краю.
- Если справа недостаточно места, он будет выровнен по правому краю.
- Если снизу достаточно места, он откроется снизу.
- Если снизу недостаточно места, он откроется сверху.

<Note>
Если вы пытаетесь использовать этот компонент, и кнопка «Купить» не работает, 
убедитесь, что у вас правильно настроен `projectId` в ваших провайдерах.

Если история транзакций не загружается, убедитесь, что у вас правильно настроен 
`apiKey` в ваших провайдерах.
</Note>


## Настройка `WalletIsland` и расширенного `Wallet`

Испытайте магию, просто вставив компоненты `<Wallet />` и расширенного `Wallet`, 
и наблюдайте, как ваш компонент плавно оживает.

Как и [`WalletDefault`](/onchainkit/wallet/wallet) , `WalletAdvancedDefault` использует 
несколько компонентов [`<Identity>`](/onchainkit/identity/identity), таких как [`<Avatar>`](/onchainkit/identity/avatar),
[`<Name>`](/onchainkit/identity/name), и [`<Address>`](/onchainkit/identity/address).

We've introduces several advanced wallet components, including
`<WalletAdvancedAddressDetails>`, `<WalletAdvancedTokenHoldings>`, and
`<WalletAdvancedTransactionActions>`.

```tsx
import {
  ConnectWallet,
  Wallet,
  WalletDropdown,
  WalletAdvancedAddressDetails,
  WalletAdvancedTokenHoldings,
  WalletAdvancedTransactionActions,
  WalletAdvancedWalletActions,
} from '@coinbase/onchainkit/wallet';
import { Address, Avatar, Name, Identity } from '@coinbase/onchainkit/identity';
import { color } from '@coinbase/onchainkit/theme';

export function YourWalletAdvanced() {
  return (
    <div className="flex justify-end">
      <Wallet>
        <ConnectWallet>
          <Avatar className="h-6 w-6" />
          <Name />
        </ConnectWallet>
        <WalletDropdown>
          <WalletAdvancedWalletActions />
          <WalletAdvancedAddressDetails />
          <WalletAdvancedTransactionActions />
          <WalletAdvancedTokenHoldings />
        </WalletDropdown>
      </Wallet>
    </div>
  );
}
```

При кастомизации вашей расширенной реализации `Wallet` используйте свойство `draggable` на 
`Wallet`, чтобы включить возможность перетаскивания. `draggable` по умолчанию
имеет значение `false`, но когда `draggable` установлено в `true`, вы также можете установить свойство
`draggableStartingPosition`, чтобы указать начальную позицию вашего `WalletIsland`..

```tsx
import {
  ConnectWallet,
  Wallet,
  WalletDropdown,
  WalletAdvancedAddressDetails,
  WalletAdvancedTokenHoldings,
  WalletAdvancedTransactionActions,
  WalletAdvancedWalletActions,
} from '@coinbase/onchainkit/wallet';
import { Address, Avatar, Name, Identity } from '@coinbase/onchainkit/identity';
import { color } from '@coinbase/onchainkit/theme';

export function DraggableWalletAdvanced() {
  return (
    <div className="flex justify-end">
      <Wallet
        draggable={true} // [!code focus]
        draggableStartingPosition={{ // [!code focus]
          x: window.innerWidth - 300, // [!code focus]
          y: window.innerHeight - 100, // [!code focus]
        }}
      >
        <ConnectWallet>
          <Avatar className="h-6 w-6" />
          <Name />
        </ConnectWallet>
        <WalletDropdown>
          <WalletAdvancedWalletActions />
          <WalletAdvancedAddressDetails />
          <WalletAdvancedTransactionActions />
          <WalletAdvancedTokenHoldings />
        </WalletDropdown>
      </Wallet>
    </div>
  );
}
```

## Компоненты

Компоненты предназначены для совместной иерархической работы. Для каждого компонента убедитесь в следующем:

- `<Wallet />` - Служит основным контейнером для всех компонентов, связанных с кошельком.
- `<WalletDropdown />` - Содержит дополнительную информацию и опции кошелька. Размещайте внутри компонента `<Wallet />`.
- `<WalletAdvancedWalletActions />` - Предоставляет действия с кошельком, такие как просмотр истории транзакций, просмотр QR-кода и отключение кошелька. Размещайте внутри компонента `<WalletDropdown />`.
- `<WalletAdvancedAddressDetails />` - Отображает адрес пользователя, аватар и баланс портфеля в фиате. Размещайте внутри компонента `<WalletDropdown />`.
- `<WalletAdvancedTransactionActions />` - Кнопки для покупки криптовалюты за фиат, перевода криптовалюты и обмена (swap). Размещайте внутри компонента `<WalletDropdown />`.
- `<WalletAdvancedTokenHoldings />` - Отображает балансы токенов и их стоимость в фиате. Размещайте внутри компонента `<WalletDropdown />`.
- `<ConnectWallet />` - Обрабатывает процесс подключения кошелька. Размещайте дочерние компоненты внутри, чтобы настроить внешний вид кнопки подключения.
- `<AppWithWalletModal />` - Включает опыт агрегации кошельков.
- `<Identity />` - Отображает информацию об идентификации пользователя. Размещайте внутри `<WalletDropdown />` для полного представления профиля.
- `<WalletDropdownBasename />` - Отображает Basename пользователя внутри выпадающего списка.
- `<WalletDropdownLink />` - Создает пользовательскую ссылку внутри выпадающего списка. Используйте свойство `icon` для добавления иконки и `href` для указания назначения.
- `<WalletDropdownDisconnect />` - Предоставляет опцию отключения внутри выпадающего списка.

Дополнительные компоненты для настройки интерфейса кошелька включают:

- `<Avatar />` - Отображает изображение аватара пользователя.
- `<Name />` - Показывает имя пользователя или ENS.
- `<Badge />` - Может использоваться для отображения дополнительного статуса или информации о пользователе.
- `<Address />` - Показывает адрес кошелька пользователя.
- `<EthBalance />` - Отображает баланс ETH пользователя.

Компонент Wallet автоматически обрабатывает состояние подключения кошелька и соответствующим образом обновляет пользовательский интерфейс.
Вам нужно обернуть ваше приложение или соответствующую его часть этими компонентами, 
чтобы обеспечить полноценный опыт взаимодействия с кошельком.
