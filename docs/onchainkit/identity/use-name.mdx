title: useName
---


Хук `useName` используется для получения имени от провайдера 
ончейн-идентификации по заданному адресу.

Он реализован с помощью [useQuery](https://tanstack.com/query/latest/docs/framework/react/reference/useQuery) из `@tanstack/react-query` и возвращает объект `UseQueryResult`, позволяя вам передавать все опции `@tanstack/react-query`.

## Использование

Получение имени ENS по адресу:

<CodeGroup>
```tsx code
import { useName } from '@coinbase/onchainkit/identity';

const address = '0x02feeb0AdE57b6adEEdE5A4EEea6Cf8c21BeB6B1';
const { data: name, isLoading } = await useName({ address });
```

```ts return value
{ data: 'zizzamia.eth', isLoading: false }
```
</CodeGroup>

Получение Basename по адресу:

<CodeGroup>
```tsx code
import { useName } from '@coinbase/onchainkit/identity';
import { base } from 'viem/chains';

const address = '0x02feeb0AdE57b6adEEdE5A4EEea6Cf8c21BeB6B1';
const { data: name, isLoading } = await useName({ address, chain: base });
```

```ts return value
{ data: 'zizzamia.base.eth', isLoading: false }
```
</CodeGroup>
