`Identity` - это провайдер контекста React, который организует компоненты идентификации.

## Использование

Показать аватар пользователя, имя с аттестацией и адрес:

```tsx
import { Avatar, Identity, Name, Badge, Address } from '@coinbase/onchainkit/identity';

<Identity
  address="0x838aD0EAE54F99F1926dA7C3b6bFbF617389B4D9"
  schemaId="0xf8b05c79f090979bf4a80270aba232dff11a10d9ca55c4f88de95317970f0de9"
>
  <Avatar />
  <Name>
    <Badge />
  </Name>
  <Address />
</Identity>
```

### Бейдж с подсказкой (Tooltip)

Вы можете включить подсказку для бейджа аттестации, чтобы предоставить контекст о том, что представляет собой бейдж:

```tsx
import { Avatar, Identity, Name, Badge, Address } from '@coinbase/onchainkit/identity';

<Identity
  address="0x838aD0EAE54F99F1926dA7C3b6bFbF617389B4D9"
  schemaId="0xf8b05c79f090979bf4a80270aba232dff11a10d9ca55c4f88de95317970f0de9"
>
  <Avatar />
  <Name> 
    <Badge 
      tooltip={true} // [!code focus]
    />
  </Name>
  <Address />
</Identity>
```
