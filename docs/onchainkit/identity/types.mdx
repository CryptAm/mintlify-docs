## `AddressReact`

```ts
type AddressReact = {
  address?: Address | null; // Ethereum-адрес для отображения.
  className?: string; // Опциональный className для переопределения стилей верхнего элемента span.
  isSliced?: boolean; // Определяет, должен ли отображаемый адрес быть сокращенным.
  hasCopyAddressOnClick?: boolean; // По умолчанию true. Опциональный boolean для отключения функции копирования адреса при клике.
};
```

## `Attestation`

```ts
type Attestation = {
  attester: Address; // Аттестатор, создавший аттестацию.
  decodedDataJson: string; // Данные аттестации, декодированные в JSON.
  expirationTime: number; // Unix-время истечения срока действия аттестации (0 означает отсутствие срока).
  id: string; // Уникальный идентификатор аттестации.
  recipient: Address; // Ethereum-адрес получателя аттестации.
  revocationTime: number; // Unix-время отзыва аттестации, если применимо.
  revoked: boolean; // Boolean, указывающий, является ли аттестация отзывной или нет.
  schemaId: EASSchemaUid; // Идентификатор схемы, связанной с аттестацией.
  time: number; // Unix-время создания аттестации.
};
```

## `AvatarReact`

```ts
type AvatarReact = {
  address?: Address | null; // Ethereum-адрес, для которого нужно получить аватар.
  chain?: Chain; // Опциональная цепь для разрешения домена
  className?: string; // Опциональный className для переопределения стилей верхнего элемента div.
  loadingComponent?: JSX.Element; // Опциональный пользовательский компонент для отображения во время загрузки данных аватара.
  defaultComponent?: JSX.Element; // Опциональный пользовательский компонент для отображения, когда имя ENS или аватар недоступны.
  children?: ReactNode; // Опциональная аттестация путем передачи компонента Badge в качестве дочернего элемента
} & ImgHTMLAttributes<HTMLImageElement>; // Опциональные дополнительные атрибуты изображения для применения к аватару.
```

## `BadgeReact`

```ts
type BadgeReact = {
  className?: string; // Опциональный className для переопределения стилей верхнего элемента span.
  tooltip?: boolean | string; // Управляет отображением подсказки при наведении на бейдж. Когда `true`, подсказка отображает название аттестации. Когда указана строка, этот текст переопределяет отображение по умолчанию. По умолчанию `false`.
};
```
