OnchainKit - это ваш основной SDK для создания красивых ончейн-приложений. Выпускайте за минуты, а не недели.

Любой может создать ончейн-приложение за 15 минут с помощью OnchainKit. Опыт работы с блокчейном не требуется.

## Почему OnchainKit?

OnchainKit упрощает разработку приложений, предоставляя комплексный набор инструментов, который сочетает мощные ончейн-функции с удобным для разработчика дизайном:

- **Эргономичный дизайн:** Полнофункциональные инструменты, которые делают сложные ончейн-взаимодействия интуитивно понятными
- **Проверенные паттерны:** Отраслевые лучшие практики, упакованные в готовые к использованию решения
- **Специализированные компоненты:** Предварительно собранные модули для распространенных ончейн-сценариев
- **Независимость от фреймворка:** Совместимость с любым фреймворком, поддерживающим React
- **Усилен Base:** Глубокая интеграция с протокольными функциями и экосистемой Base

Мы рекомендуем начинать новое OnchainKit-приложение с помощью `create onchain`, который автоматически настроит всё за вас. Чтобы создать проект, выполните:

```bash Terminal
npm create onchain@latest
```

После запросов `create onchain` создаст папку с названием вашего проекта и установит необходимые зависимости.
