---
title: useBuildWithdrawFromMorphoTx
---


Хук `useBuildWithdrawFromMorphoTx` используется для построения транзакции вывода (withdraw) из Morpho из React-компонента.

Хук обрабатывает следующее:

- Получение данных, необходимых для построения транзакции
- Использование [`buildWithdrawFromMorphoTx`](/onchainkit/api/build-withdraw-from-morpho-tx) для построения транзакции
- Возврат вызовов транзакции (transaction calls) для использования с компонентом `<Transaction />`

## Использование

```tsx code
import { useBuildWithdrawFromMorphoTx } from '@coinbase/onchainkit/earn';

const { calls } = useBuildWithdrawFromMorphoTx({
  vaultAddress: '0x...',
  recipientAddress: '0x...',
  amount: '1000000000000000000',
  tokenDecimals: 18,
});
```

[`{ calls: Call[] }`](/onchainkit/transaction/types#calls)
