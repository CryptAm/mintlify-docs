---
title: useEarnContext
---


Хук `useEarnContext` используется для доступа к значениям контекста компонента `Earn`.

Его можно использовать для создания полностью пользовательских интерфейсов депозита и вывода (withdraw), и он содержит все соответствующие данные о Morpho vault.


## Использование

Примечание: Этот хук должен использоваться внутри компонента `<Earn />` или `<EarnProvider />` (headless-версия).

```tsx code
import { useEarnContext } from '@coinbase/onchainkit/earn';

const { depositAmount, 
        setDepositAmount, 
        apy, 
        nativeApy, 
        rewards, 
        vaultFee, 
        vaultToken, 
        liquidity, 
        deposits 
       // ...  
        } = useEarnContext();

// Используйте значения для создания пользовательского интерфейса депозита!
```
