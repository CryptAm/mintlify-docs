Компонент `NFTMintCard` предоставляет простой способ создания NFT. Просто введите адрес контракта NFT и идентификатор токена (для контрактов ERC1155) и включите подкомпоненты, которые вы хотите отобразить.

## Предварительные требования

Перед использованием компонента `NFTMintCard` убедитесь, что вы выполнили шаги [Getting Started](/onchainkit/getting-started).

Чтобы использовать компонент `NFTMintCard`, вам необходимо предоставить API-ключ в `OnchainKitProvider`. Вы можете получить его, следуя нашим шагам [Getting Started](/onchainkit/getting-started#get-your-public-api-key).

### Создание нового проекта

Если вы начинаете новый проект, мы рекомендуем использовать `create onchain` для создания структуры вашего проекта.

```bash
npm create onchain@latest
```

### Добавление в существующий проект

Если вы добавляете `NFTMintCard` в существующий проект, просто установите OnchainKit.

<CodeGroup>
```bash npm
npm install @coinbase/onchainkit
```

```bash yarn
yarn add @coinbase/onchainkit
```

```bash pnpm
pnpm add @coinbase/onchainkit
```

```bash bun
bun add @coinbase/onchainkit
```
</CodeGroup>

Оберните `<OnchainKitProvider />` вокруг вашего приложения, следуя шагам в [Getting Started](/onchainkit/getting-started#add-providers).

## Быстрый старт

Компонент `NFTMintCardDefault` является упрощенной версией компонента `NFTMintCard`, предназначенной для упрощения процесса интеграции для разработчиков. Вместо ручного определения каждого подкомпонента разработчики могут использовать эту сокращенную версию, которая отображает нашу рекомендуемую реализацию компонента и включает `NFTCreator`, `NFTMedia`, `NFTCollectionTitle`, `NFTQuantitySelector`, `NFTAssetCost` и `NFTMintButton`.

```tsx
import { NFTMintCardDefault } from '@coinbase/onchainkit/nft'; // [!code focus]

<NFTMintCardDefault  // [!code focus]
  contractAddress='0xb4703a3a73aec16e764cbd210b0fde9efdab8941'  // [!code focus]
/>  // [!code focus]
```

Если вы хотите больше кастомизации, вы можете выполнить следующие шаги, чтобы настроить, какие подкомпоненты отображаются.

## Не уверены, что именно создавать?

Вы можете создать свой собственный NFT для минта в [Coinbase Wallets create a Mint flow](https://wallet.coinbase.com/home/create). Просто следуйте инструкциям, чтобы создать ваш NFT, а затем скопируйте адрес контракта из URL в NFTMintCard.

## Совместимость

Компонент минта использует пользовательскую реализацию `buildMintTransaction`, которая поддерживает минты Coinbase, а также [`these supported platforms on reservoir`](https://docs.reservoir.tools/docs/minting#platform-support). Если ваш контракт не поддерживается, пожалуйста, следуйте инструкциям [`bring your own data instructions`](#bring-your-own-data) ниже.

## Расширенное использование

### Используйте свои собственные данные

Реализация `NFTMintCard` по умолчанию использует Coinbase Developer Platform для предоставления данных и требует API-ключа. Вы можете использовать свойство `useNFTData` для передачи хука, который получает данные NFT из источника по вашему выбору. Пока этот хук возвращает объект с той же структурой, что и тип [`NFTData`](/onchainkit/mint/types#nftdata), NFTMintCard будет отображаться.

Вы также можете использовать пользовательскую функцию `buildMintTransaction` для создания своей собственной транзакции минта.

```tsx
// @noErrors: 2739
import { NFTMintCard } from '@coinbase/onchainkit/nft';
import { NFTMedia } from '@coinbase/onchainkit/nft/view';
import { NFTCollectionTitle, NFTMintButton } from '@coinbase/onchainkit/nft/mint';
// ---cut-before---

function useNFTData() {  // [!code focus]
  return {  // [!code focus]
    title: 'My NFT',  // [!code focus]
    imageUrl: 'https://example.com/image.png',  // [!code focus]
  }  // [!code focus]
}  // [!code focus]

async function buildMintTransaction() {  // [!code focus]
  const response = await fetch('https://api.minttransaction.com');  // [!code focus]
  return await response.json();  // [!code focus]
}  // [!code focus]

export default function NFTComponent() {
  return (
    <NFTMintCard
      contractAddress='0xb4703a3a73aec16e764cbd210b0fde9efdab8941'
      useNFTData={useNFTData}  // [!code focus]
      buildMintTransaction={buildMintTransaction}  // [!code focus]
    >
      <NFTMedia/>
      <NFTCollectionTitle />
      <NFTMintButton disabled={true}/>
    </NFTMintCard>
  );
}
```

### Отслеживание жизненного цикла компонента

Вы можете использовать наш NFT [`LifecycleStatus`](/onchainkit/mint/types#lifecyclestatus) и свойство `onStatus` для отслеживания состояний транзакции.

NFTMintCard имеет 6 состояний жизненного цикла:

- `init` - Компонент инициализирован
- `error` - Компонент столкнулся с ошибкой
- `mediaLoading` - Медиа загружается
- `mediaLoaded` - Медиа загружено
- `transactionPending` - Транзакция минта ожидает выполнения
- `transactionLegacyExecuted` - Транзакция минта была выполнена
- `success` - Транзакция минта была успешной

```tsx
import { NFTMintCard } from '@coinbase/onchainkit/nft';
import { NFTMedia } from '@coinbase/onchainkit/nft/view';
// ---cut-before---
import type { LifecycleStatus } from '@coinbase/onchainkit/nft'; // [!code focus]

const statusHandler = (status: LifecycleStatus) => { // [!code focus]
  const { statusName, statusData } = status; // [!code focus]
  switch (statusName) { // [!code focus]
    case 'success': // [!code focus]
      // обработка успеха
    case 'error': // [!code focus]
      // обработка ошибки
    default: // [!code focus]
      // обработка состояния 'init'
  } // [!code focus]
} // [!code focus]

<NFTMintCard  // [!code focus]
  onStatus={statusHandler}  // [!code focus]
  contractAddress='0xb4703a3a73aec16e764cbd210b0fde9efdab8941'
>
  <NFTMedia />
</NFTMintCard>
// ---cut-after---
```

## Примеры использования

- **Создайте галерею ваших NFT для минта:** Создайте галерею NFT, которые вы создали для пользователей, чтобы они могли сминтить
- **Сминченный внутриигровой аватар:** Позвольте вашим пользователям создать, а затем просмотреть сминтинный аватар

## Компоненты

Компонент `NFTMintCard` может быть кастомизирован с помощью следующих компонентов:

- `<NFTCreator />` - Создатель NFT.
- `<NFTMedia square={boolean}/>` - Медиа для NFT, включает поддержку изображений, видео и аудио NFT.
- `<NFTCollectionTitle />` - Название коллекции NFT.
- `<NFTQuantitySelector />` - Селектор количества для NFT.
- `<NFTAssetCost />` - Стоимость NFT в нативной валюте и USD.
- `<NFTMintButton />` - Кнопка минта для NFT.
