## Что такое Base Appchains?

Base Appchains - это **специфичные для приложений роллапы третьего уровня (Layer 3)**, построенные на Base, которые предоставляют выделенное пространство блоков для отдельных приложений. Вместо того чтобы конкурировать с тысячами других приложений за сетевые ресурсы, вы получаете свой собственный высокопроизводительный блокчейн, который роллапится в Base и наследует безопасность Ethereum.

<Note>
Appchains третьего уровня (Layer 3) роллапятся в Base (уровень 2), который финализируется на Ethereum, предоставляя вам выделенную производительность при сохранении гарантий безопасности экосистемы Ethereum.
</Note>

Подумайте об этом как о разнице между **совместным использованием шоссе в час пик и наличием вашей собственной выделенной полосы**. При общем пространстве блоков производительность вашего приложения зависит от активности во всей сети. С Appchain вы получаете стабильную, предсказуемую производительность независимо от того, что происходит где-либо еще.

## Почему выбирают Base Appchains?

### Высокоскоростная производительность, созданная для масштабирования

Перестаньте позволять перегрузке сети влиять на пользовательский опыт. Base Appchains обеспечивают **время создания блока в 1 секунду и вывод средств менее чем за 10 секунд**, что делает их в 10 раз быстрее, чем типичные взаимодействия с блокчейном.

<Tip>
Игровые приложения, такие как Blocklords, обработали более 80 миллионов транзакций через 1,8 миллиона кошельков, используя Base Appchains, достигнув масштаба, необходимого для их игровой экосистемы, без деградации производительности.
</Tip>

### Предсказуемые, рентабельные операции

Замените непредсказуемые затраты на газ на **фиксированную ежемесячную цену**. Обрабатывайте транзакции за доли цента, устраняя необходимость спонсировать затраты на газ для ваших пользователей.

## Техническая архитектура

Base Appchains построены на **фреймворке OP Enclave**, обеспечивая быстрый вывод средств и бесшовную интеграцию с Base Mainnet. Эта архитектура позволяет осуществлять почти мгновенные мостовые переводы (bridging), сохраняя безопасность через инновационные механизмы доказательства.

<AccordionGroup>
<Accordion title="Фреймворк OP Enclave">
Построен на новейшей технологии Optimism для **почти мгновенных мостовых переводов** между вашим Appchain и Base Mainnet. Пользователи могут перемещать средства за секунды, а не за типичные 7-дневные периоды вывода традиционных роллапов.
</Accordion>

<Accordion title="Альтернативное обеспечение данных (Data Availability)">
Использует **Amazon S3** для экономичного хранения данных, сохраняя безопасность через проверку **AWS Nitro Enclave**. Этот подход значительно снижает затраты, обеспечивая целостность и доступность данных.
</Accordion>

<Accordion title="Пользовательские разрешения и токены для газа">
Контролируйте, какие контракты могут быть вызваны в вашей цепи, эффективно управляя распределением пространства блоков. Реализуйте **пользовательские токены для газа** и **системы разрешений**, защищая пользователей от цензуры через гарантированные каналы для депозитов.
</Accordion>

<Accordion title="Немедленное доказательство состояния (State Proving)">
В отличие от традиционных роллапов, которые полагаются на периоды оспаривания (challenge periods), Base Appchains используют **немедленное доказательство состояния** через безопасные анклавы (enclaves), обеспечивая мгновенную финализацию (finality) и более быстрый пользовательский опыт.
</Accordion>
</AccordionGroup>

## Сценарии использования и истории успеха

Base Appchains обеспечивают работу приложений в игровой, DeFi и корпоративной сферах, требующих высокой производительности и предсказуемых затрат.

<Tabs>
<Tab title="Игры">

**История успеха**: Super Champs выбрали Base Appchains для стабильной пропускной способности, сравнив опыт с "играми на iOS" - плавными, предсказуемыми и надежными.
</Tab>

<Tab title="DeFi">
Обрабатывайте высокочастотную торговлю, фарм доходности и сложные финансовые операции со стабильными, недорогими транзакциями, которые не колеблются от сетевой активности.

**История успеха**: Приложения, обрабатывающие данные о торговле с высоким объемом, выигрывают от предсказуемых затрат и выделенной пропускной способности для чувствительных ко времени операций.
</Tab>

<Tab title="Корпоративные решения">
Развертывайте готовые к соответствию нормативным требованиям (compliance-ready) решения с выделенной инфраструктурой, пользовательскими разрешениями и возможностью управления контролем доступа, сохраняя прозрачность.

**История успеха**: Proof 8 использует технологию блокчейна для верифицируемого владения инвентарем на складах и винокурнях, где корпоративные клиенты отдают приоритет производительности, безопасности и конфиденциальности.
</Tab>
</Tabs>

<Warning>
Base Appchains предназначены для **зрелых проектов** со значительным объемом транзакций. Если вы только начинаете или имеете низкий объем транзакций, рассмотрите сначала разработку на Base Mainnet, чтобы установить соответствие продукта рынку.
</Warning>

## Когда стоит рассмотреть Appchain?

Base Appchains идеально подходят для приложений, которые переросли ограничения общего пространства блоков. Используйте этот чек-лист, чтобы определить, подходит ли Appchain для вашего проекта:

<Check>
**Высокий объем транзакций**: Ваше приложение генерирует тысячи транзакций ежедневно, и производительность страдает от перегрузки сети
</Check>

<Check>
**Значительное спонсирование газа**: Вы тратите существенные суммы на спонсирование затрат на газ для пользователей через Paymaster или аналогичные решения
</Check>

<Check>
**Критичные к производительности операции**: Пользовательский опыт негативно страдает от переменного времени транзакций или перегрузки сети
</Check>

<Check>
**Пользовательские требования**: Вам нужны пользовательские токены для газа, специализированные разрешения или механизмы управления, недоступные в общих цепях
</Check>

<Check>
**Предсказуемые затраты**: Фиксированные операционные затраты важны для вашей бизнес-модели и бюджетирования
</Check>

<Info>
Если вы рассматриваете возможность запуска собственного блокчейна L1 или L2, Base Appchains предлагают убедительную альтернативу с более быстрым выходом на рынок, проверенной инфраструктурой и немедленным доступом к экосистеме Base.
</Info>

## Часто задаваемые вопросы

<AccordionGroup>
<Accordion title="Как Base Appchains сравниваются с запуском собственного L1 или L2?">
Base Appchains предлагают значительные преимущества по сравнению с запуском независимой инфраструктуры блокчейна:

- **Более быстрый выход на рынок**: Развертывание за недели, а не месяцы или годы
- **Проверенная инфраструктура**: Построена на испытанной технологии Base и OP Stack
- **Немедленный доступ к экосистеме**: Пользователи и ликвидность из Base Mainnet
- **Меньшая операционная нагрузка**: Полностью управляемая инфраструктура и обслуживание
- **Совместимость с Ethereum**: Наследование безопасности без пользовательских наборов валидаторов или механизмов консенсуса
</Accordion>

<Accordion title="Каков пользовательский опыт онбординга на мой Appchain?">
Base Appchains обеспечивают **бесшовный онбординг**, аналогичный приложениям на Base Mainnet:

- Пользователи могут переводить средства между Base и вашим Appchain за **секунды**
- Тот же опыт работы с кошельком по всей экосистеме Base
- Интеграция Smart Wallet для абстракции аккаунтов (account abstraction)
- Знакомые паттерны транзакций и интерфейсы
</Accordion>

<Accordion title="Контролирую ли я инфраструктуру моего Appchain?">
**Coinbase управляет основной инфраструктурой** от вашего имени, включая:

- Работу и обслуживание секвенсора
- Инфраструктуру нод и обновления
- Управление ключами безопасности
- Системы мониторинга и оповещений

Вы сохраняете контроль над **конфигурациями на уровне приложения**, такими как токены для газа, разрешения и управление (governance), получая при этом преимущества управления инфраструктурой корпоративного уровня.
</Accordion>

<Accordion title="Как работают мостовые переводы между Base и моим Appchain?">
Base Appchains используют **фреймворк OP Enclave** для **почти мгновенных мостовых переводов**:

- Перемещайте средства между Base и вашим Appchain за секунды
- Нет 7-дневных периодов вывода, как у традиционных роллапов
- Сохраняет безопасность через криптографические доказательства
- Бесшовный пользовательский опыт между цепями
</Accordion>

<Accordion title="А как насчет децентрализации и устойчивости к цензуре?">
Base Appchains балансируют между **операционной эффективностью** и **устойчивостью к цензуре**:

- Пользовательские разрешения контролируют высокопроизводительные операции
- **Гарантированные каналы для депозитов** предотвращают цензуру через прямые депозиты в Base
- У пользователей всегда есть возможность обжалования через мост Base
- Наследует долгосрочную дорожную карту децентрализации Ethereum
</Accordion>
