Добавьте провайдер и инициализируйте MiniKit на вашей главной странице.

## Добавьте MiniKitProvider

Создайте `providers/MiniKitProvider.tsx` и оберните `app/layout.tsx`.

```jsx providers/MiniKitProvider.tsx
'use client';
import { MiniKitProvider } from '@coinbase/onchainkit/minikit';
import { ReactNode } from 'react';
import { base } from 'wagmi/chains';

export function MiniKitContextProvider({ children }: { children: ReactNode }) {
  return (
    <MiniKitProvider apiKey={process.env.NEXT_PUBLIC_CDP_CLIENT_API_KEY} chain={base}>
      {children}
    </MiniKitProvider>
  );
}
```

Оберните ваш корневой макет (layout):

```jsx app/layout.tsx
import { MiniKitContextProvider } from '@/providers/MiniKitProvider';

export default function RootLayout({ children }: { children: React.ReactNode }) {
  return (
    <html lang='en'>
      <body>
        <MiniKitContextProvider>{children}</MiniKitContextProvider>
      </body>
    </html>
  );
}
```

## Инициализируйте MiniKit на вашей странице

Используйте `useMiniKit()` для вызова `setFrameReady()`, когда ваше приложение готово.

```jsx app/page.tsx
'use client';
import { useEffect } from 'react';
import { useMiniKit } from '@coinbase/onchainkit/minikit';

export default function HomePage() {
  const { setFrameReady, isFrameReady } = useMiniKit();

  useEffect(() => {
    if (!isFrameReady) setFrameReady();
  }, [isFrameReady, setFrameReady]);

  return <div>Your app content goes here</div>;
}
```

<Tip>
Провайдер настраивает wagmi и react‑query и использует коннектор Farcaster, когда он доступен.
</Tip>
