---
title: Добавьте метаданные Frame
description: Определите метаданные fc:frame для отображения расширенных эмбедов с кнопками запуска
---

Метаданные критически важны для обнаружения вашего приложения. Они позволяют создавать расширенные эмбеды при публикации в социальной ленте и обеспечивают правильную индексацию. Добавьте метаданные `fc:frame`, чтобы при публикации ссылок отображался эмбед с кнопкой запуска.

<Warning>
Поместите метатег в `<head>` и убедитесь, что все используемые ассеты (assets) используют HTTPS.
</Warning>

### Next.js (generateMetadata)

```ts app/layout.tsx
export async function generateMetadata(): Promise<Metadata> {
  const URL = process.env.NEXT_PUBLIC_URL as string;
  return {
    title: process.env.NEXT_PUBLIC_ONCHAINKIT_PROJECT_NAME,
    description: 'Generated by `create-onchain --mini`',
    other: {
      'fc:frame': JSON.stringify({
        version: 'next',
        imageUrl: process.env.NEXT_PUBLIC_APP_HERO_IMAGE,
        button: {
          title: `Launch ${process.env.NEXT_PUBLIC_ONCHAINKIT_PROJECT_NAME}`,
          action: {
            type: 'launch_frame',
            name: process.env.NEXT_PUBLIC_ONCHAINKIT_PROJECT_NAME,
            url: URL,
            splashImageUrl: process.env.NEXT_PUBLIC_SPLASH_IMAGE,
            splashBackgroundColor: process.env.NEXT_PUBLIC_SPLASH_BACKGROUND_COLOR,
          },
        },
      }),
    },
  };
}
```
