## Форкните, кастомизируйте и задеплойте

Форк референсного мини-приложения может сэкономить время, после чего вы можете добавить ваши конкретные потоки и брендинг.

<Steps>
  <Step title='Форкните подходящий стартовый проект'>
    Выберите пример, близкий к вашему случаю использования. Обновите файл
    `/.well-known/farcaster.json`, метаданные в `app/layout.tsx` и 
    переменные окружения.
  </Step>
  <Step title='Кастомизируйте UI и контракты'>
    Настройте токены темы, текст и любые адреса контрактов или конечные точки (endpoints), 
    требуемые для ваших потоков.
  </Step>
  <Step title='Деплойте и проверьте в Base App'>
    Деплойте с помощью Vercel, протестируйте публикацию и обнаружение, и проверьте потоки 
    кошелька и Paymaster от начала до конца.
  </Step>
</Steps>

Чтобы форкнуть референсное мини-приложение, откройте терминал и выполните следующую команду:

```bash
git clone https://github.com/base/demos.git
cd demos/minikit/three-card-monte
npm install
npm run dev
```
