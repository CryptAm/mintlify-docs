title: Манифест (CLI)
description: Сгенерируйте учетные данные для ассоциации аккаунта с помощью CLI
---

Сгенерируйте ваши учетные данные для ассоциации аккаунта Farcaster.

```bash Terminal
npx create-onchain --manifest
```

Следуйте инструкциям, чтобы подключить ваш кастодиальный кошелек Farcaster, добавить ваш задеплоенный URL и подписать. CLI запишет `FARCASTER_HEADER`, `FARCASTER_PAYLOAD`, и `FARCASTER_SIGNATURE` в ваш `.env`.


<Warning>
Во время тестирования установите `noindex: true` в вашем манифесте, чтобы избежать индексации.
</Warning>
