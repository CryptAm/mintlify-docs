Установите MiniKit (часть OnchainKit) в ваш существующий проект Next.js с App Router.

## Предварительные требования

<AccordionGroup>
<Accordion title="Next.js App Router">
Ваш проект использует директорию `app/` (App Router).
</Accordion>
<Accordion title="Деплой">
Ваше приложение задеплоено и публично доступно по HTTPS (например, Vercel).
</Accordion>
<Accordion title="Аккаунт Farcaster">
У вас есть доступ к вашему кастодиальному кошельку Farcaster для подписания манифеста.
</Accordion>
<Accordion title="Аккаунт CDP (для API ключа)">
Войдите в Coinbase Developer Platform, чтобы получить Client API ключ.
</Accordion>
</AccordionGroup>

## Установите зависимости

```bash
npm install @coinbase/onchainkit
```

<Check>
Убедитесь, что `@coinbase/onchainkit` появился в вашем `package.json`.
</Check>
