# Ресурсы для разработчиков

При создании мини-приложения для Base App (TBA) и Farcaster есть пять основных источников документации, к которым вы будете возвращаться снова и снова. Это не просто документация - это инструменты, которые помогут вам решать проблемы, исправлять ошибки и продолжать разработку.

Не волнуйтесь, если сначала они кажутся пугающими. От вас не ожидается их запоминание. Вместо этого вы научитесь обращаться к этим документам и искать в них по мере необходимости - и давать промты вашему AI ассистенту, чтобы помочь понять и использовать их. (Мы покажем, как это сделать, в следующем разделе.)

Вот ключевая идея:

- **Base Docs** фокусируются на цепи Base и платформе приложений - всё, от MiniKit до OnchainKit и инструментов для смарт-аккаунтов.
- **Coinbase Developer Platform (CDP)** предоставляет доступ к инструментам инфраструктуры - API кошельков, Paymaster и фиатные онрампы. Считайте это слоем бэкенд-сервисов для вашего приложения.

## Пример промта для понимания ключевых инструментов

Используйте промты, подобные приведенному ниже, чтобы помочь AI ассистенту объяснить концепции из документации:

```
Помогите мне понять фронтенд разработку для блокчейна, объяснив эти концепции простыми словами:

  1. Что такое Wagmi и как он упрощает подключение кошельков в React?
  2. Как Viem обрабатывает низкоуровневые операции с блокчейном?
  3. В чем разница между чтением данных из блокчейна и записью транзакций?
  4. Как эти инструменты работают вместе в типичном мини-приложении?

Сосредоточьтесь на сети Base и предоставьте практические примеры для каждой концепции. Включите примеры кода, где это полезно.
```

В следующем разделе мы покажем, как создавать промты для AI инструментов, чтобы разобраться во всей этой документации - чтобы даже сложные вещи стало легче использовать.
