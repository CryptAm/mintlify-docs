```html title="index.html" lineNumbers

              }

        }
      };

      // –ü–ª–∞—Ç–µ–∂ USDC –≤ –æ–¥–Ω–æ –Ω–∞–∂–∞—Ç–∏–µ —Å –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ–º window.base API (—Ä–∞–±–æ—Ç–∞–µ—Ç —Å –≤—Ö–æ–¥–æ–º –∏–ª–∏ –±–µ–∑)
```
