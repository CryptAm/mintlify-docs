---
title: "eth_getTransactionCount"
description: "Получить количество транзакций, отправленных с адреса"
---

Определено в [Спецификации Ethereum JSON-RPC](https://ethereum.org/en/developers/docs/apis/json-rpc/)

<Info>
Возвращает количество транзакций, отправленных с указанного адреса.
</Info>

## Параметры

<ParamField body="address" type="string" required>
Адрес, для которого требуется получить количество транзакций (20 байт).
</ParamField>

<ParamField body="blockParameter" type="string" required>
Номер блока в виде целого числа или строки "latest", "earliest" или "pending".
</ParamField>

## Возвращает

<ResponseField name="result" type="string">
Шестнадцатеричная строка (hex), представляющая целое число количества транзакций, отправленных с этого адреса.
</ResponseField>

<RequestExample>
```json Request
{
  "id": 1,
  "jsonrpc": "2.0",
  "method": "eth_getTransactionCount",
  "params": [
    "0x407d73d8a49eeb85d32cf465507dd71d507100c1",
    "latest"
  ]
}
```
</RequestExample>

<ResponseExample>
```json Success Response
{
  "id": 1,
  "jsonrpc": "2.0",
  "result": "0x1"
}
```
</ResponseExample>

## Обработка ошибок

| Код    | Сообщение                          | Описание    |
| ------ | ---------------------------------- | ----------- |
| -32602 | Invalid address or block parameter | Указан некорректный адрес или параметр блока |
| 4100   | Requested method not supported     | Метод не поддерживается кошельком |

<Info>
Это значение используется как nonce для последующих транзакций с этого адреса.
</Info>
