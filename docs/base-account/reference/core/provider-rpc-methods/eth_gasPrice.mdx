---
title: "eth_gasPrice"
description: "Получить текущую цену газа"
---

Определено в [Спецификации Ethereum JSON-RPC](https://ethereum.org/en/developers/docs/apis/json-rpc/)

<Info>
Возвращает текущую цену за единицу газа в wei.
</Info>

## Параметры

Этот метод не принимает параметров.

## Возвращает

<ResponseField name="result" type="string">
Шестнадцатеричная строка, представляющая текущую цену газа в wei.
</ResponseField>

<RequestExample>
```json Request
{
  "id": 1,
  "jsonrpc": "2.0",
  "method": "eth_gasPrice",
  "params": []
}
```
</RequestExample>

<ResponseExample>
```json Success Response
{
  "id": 1,
  "jsonrpc": "2.0",
  "result": "0x09184e72a000"
}
```
</ResponseExample>

## Обработка ошибок

| Код  | Сообщение                      | Описание    |
| ---- | ------------------------------ | ----------- |
| 4100 | Requested method not supported | Метод не поддерживается кошельком |

<Info>
Результат представлен в wei. Для конвертации в gwei разделите на 10^9.
</Info>
