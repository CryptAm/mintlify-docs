---
title: "eth_getProof"
description: "Получить Мерклово доказательство для значений аккаунта и хранилища"
---

Определено в [Спецификации Ethereum JSON-RPC](https://ethereum.org/en/developers/docs/apis/json-rpc/)

<Info>
Возвращает значения аккаунта и хранилища указанного аккаунта, включая Мерклово доказательство.
</Info>

## Параметры

<ParamField body="address" type="string" required>
Адрес аккаунта (20 байт).
</ParamField>

<ParamField body="storageKeys" type="array" required>
Массив ключей хранилища, которые должны быть проверены и включены в доказательство.
</ParamField>

<ParamField body="blockParameter" type="string" required>
Номер блока в виде целого числа или строки "latest", "earliest" или "pending".
</ParamField>

## Возвращает

<ResponseField name="result" type="object">
Объект доказательства, содержащий доказательства аккаунта и хранилища.

<Expandable title="Свойства объекта доказательства">
<ResponseField name="address" type="string">
Адрес аккаунта.
</ResponseField>

<ResponseField name="accountProof" type="array">
Массив rlp-сериализованных узлов Мерклова дерева, начиная с узла stateRoot.
</ResponseField>

<ResponseField name="balance" type="string">
Баланс аккаунта (hex).
</ResponseField>

<ResponseField name="codeHash" type="string">
Хэш кода аккаунта (hex).
</ResponseField>

<ResponseField name="nonce" type="string">
nonce аккаунта (hex).
</ResponseField>

<ResponseField name="storageHash" type="string">
SHA3 хэш StorageRoot (hex).
</ResponseField>

<ResponseField name="storageProof" type="array">
Массив объектов доказательств хранилища.

<Expandable title="Свойства доказательства хранилища">
<ResponseField name="key" type="string">
Запрошенный ключ хранилища (hex).
</ResponseField>

<ResponseField name="value" type="string">
Значение в хранилище (hex).
</ResponseField>

<ResponseField name="proof" type="array">
Массив rlp-сериализованных узлов Мерклова дерева (Merkle tree).
</ResponseField>
</Expandable>
</ResponseField>
</Expandable>
</ResponseField>

## Обработка ошибок

| Код    | Сообщение                      | Описание    |
| ------ | ------------------------------ | ----------- |
| -32602 | Invalid parameters             | Некорректный адрес, ключи хранилища или параметр блока |
| 4100   | Requested method not supported | Метод не поддерживается кошельком |

<Info>
Этот метод полезен для проверки состояний аккаунтов и значений хранилища с использованием Меркловых доказательств без необходимости доверять провайдеру.
</Info>
