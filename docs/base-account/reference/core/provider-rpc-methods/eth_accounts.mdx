Определено в [EIP-1474](https://github.com/ethereum/EIPs/blob/master/EIPS/eip-1474.md)

<Info>
Возвращает список адресов, принадлежащих подключенному кошельку. В отличие от `eth_requestAccounts`, этот метод возвращает пустой массив, если нет доступных аккаунтов или если пользователь не авторизовал ни одного аккаунта для вызывающей стороны. Этот метод не запрашивает у пользователя разрешение на подключение.
</Info>

## Параметры

Этот метод не принимает параметров.

## Возвращает

<ResponseField name="result" type="Array<string>">
Массив Ethereum-адресов (шестнадцатеричные строки), которыми управляет подключенный пользователь. Массив обычно содержит один адрес, который является выбранным в данный момент аккаунтом в кошельке. Если кошелек не подключен или аккаунты не авторизованы, этот метод возвращает пустой массив.
</ResponseField>


<RequestExample>
```json Request
{
  "id": 1,
  "jsonrpc": "2.0",
  "method": "eth_accounts",
  "params": []
}
```
</RequestExample>

<ResponseExample>
```json Connected Accounts
{
  "id": 1,
  "jsonrpc": "2.0",
  "result": ["0xabc123..."]
}
```

```json No Accounts Connected
{
  "id": 1,
  "jsonrpc": "2.0",
  "result": []
}
```
</ResponseExample>

| Код  | Сообщение                      | Описание                                                |
| ---- | ------------------------------ | ------------------------------------------------------- |
| 4100 | Requested method not supported | Провайдер не поддерживает метод `eth_accounts`         |
| 4900 | Disconnected                   | Провайдер отключен от кошелька                         |
