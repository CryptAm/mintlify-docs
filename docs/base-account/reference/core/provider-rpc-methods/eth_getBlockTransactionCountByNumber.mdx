---
title: "eth_getBlockTransactionCountByNumber"
description: "Получить количество транзакций в блоке по номеру блока"
---

Определено в [Спецификации Ethereum JSON-RPC](https://ethereum.org/en/developers/docs/apis/json-rpc/)

<Info>
Возвращает количество транзакций в блоке по его номеру.
</Info>

## Параметры

<ParamField body="blockParameter" type="string" required>
Номер блока в виде целого числа или строки "latest", "earliest" или "pending".
</ParamField>

## Возвращает

<ResponseField name="result" type="string">
Количество транзакций в этом блоке в виде шестнадцатеричной строки (hex).
</ResponseField>

<RequestExample>
```json Latest Block
{
  "id": 1,
  "jsonrpc": "2.0",
  "method": "eth_getBlockTransactionCountByNumber",
  "params": [
    "latest"
  ]
}
```

```json Specific Block
{
  "id": 1,
  "jsonrpc": "2.0",
  "method": "eth_getBlockTransactionCountByNumber",
  "params": [
    "0x1b4"
  ]
}
```
</RequestExample>

<ResponseExample>
```json Success Response
{
  "id": 1,
  "jsonrpc": "2.0",
  "result": "0xa"
}
```

```json Empty Block
{
  "id": 1,
  "jsonrpc": "2.0",
  "result": "0x0"
}
```
</ResponseExample>

## Обработка ошибок

| Код    | Сообщение                      | Описание    |
| ------ | ------------------------------ | ----------- |
| -32602 | Invalid block parameter        | Указан некорректный параметр блока |
| 4100   | Requested method not supported | Метод не поддерживается кошельком |

<Info>
Этот метод возвращает количество транзакций в конкретном блоке, идентифицированном по его номеру.
</Info>
