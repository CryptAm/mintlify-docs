---
title: "eth_getBalance"
description: "Получить баланс аккаунта на указанном блоке"
---

Определено в [Спецификации Ethereum JSON-RPC](https://ethereum.org/en/developers/docs/apis/json-rpc/)

<Info>
Возвращает баланс аккаунта по указанному адресу.
</Info>

## Параметры

<ParamField body="address" type="string" required>
Адрес для проверки баланса (20 байт).
</ParamField>

<ParamField body="blockParameter" type="string" required>
Номер блока в виде целого числа или строки "latest", "earliest" или "pending".
</ParamField>

## Возвращает

<ResponseField name="result" type="string">
Шестнадцатеричная строка (hex), представляющая текущий баланс в wei.
</ResponseField>

<RequestExample>
```json Request
{
  "id": 1,
  "jsonrpc": "2.0",
  "method": "eth_getBalance",
  "params": ["0x407d73d8a49eeb85d32cf465507dd71d507100c1", "latest"]
}
```
</RequestExample>

<ResponseExample>
```json Success Response
{
  "id": 1,
  "jsonrpc": "2.0",
  "result": "0x0234c8a3397aab58"
}
```
</ResponseExample>

## Обработка ошибок

| Код    | Сообщение                          | Описание    |
| ------ | ---------------------------------- | ----------- |
| -32602 | Invalid address or block parameter | Указан некорректный адрес или параметр блока |
| 4100   | Requested method not supported     | Метод не поддерживается кошельком |

<Info>
Результат представляет баланс в wei. Для конвертации в ETH разделите на 10^18.
</Info>
