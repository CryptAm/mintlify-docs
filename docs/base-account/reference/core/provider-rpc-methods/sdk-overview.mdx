# Обзор

## Введение

CoinbaseWalletProvider - это [JavaScript Ethereum провайдер](https://eips.ethereum.org/EIPS/eip-1193).
Он позволяет JavaScript приложениям выполнять Ethereum RPC запросы через его метод `request`.
Эти запросы обрабатываются одним из трех способов:

1. Отправляются в Кошелек (мобильное приложение, расширение или всплывающее окно).
2. Обрабатываются локально.
3. Передаются провайдеру RPC по умолчанию для данной цепи, если он существует.
