---
title: "eth_getUncleCountByBlockNumber"
description: "Получить количество дядей в блоке по номеру блока"
---

Определено в [Спецификации Ethereum JSON-RPC](https://ethereum.org/en/developers/docs/apis/json-rpc/)

<Info>
Возвращает количество дядей в блоке по его номеру.
</Info>

## Параметры

<ParamField body="blockParameter" type="string" required>
Номер блока в виде целого числа или строки "latest", "earliest" или "pending".
</ParamField>

## Возвращает

<ResponseField name="result" type="string">
Количество дядей в этом блоке в виде шестнадцатеричной строки (hex).
</ResponseField>

<RequestExample>
```json Latest Block
{
  "id": 1,
  "jsonrpc": "2.0",
  "method": "eth_getUncleCountByBlockNumber",
  "params": [
    "latest"
  ]
}
```

```json Specific Block
{
  "id": 1,
  "jsonrpc": "2.0",
  "method": "eth_getUncleCountByBlockNumber",
  "params": [
    "0x1b4"
  ]
}
```
</RequestExample>

<ResponseExample>
```json Success Response
{
  "id": 1,
  "jsonrpc": "2.0",
  "result": "0x1"
}
```

```json No Uncles
{
  "id": 1,
  "jsonrpc": "2.0",
  "result": "0x0"
}
```
</ResponseExample>

## Обработка ошибок

| Код    | Сообщение                      | Описание    |
| ------ | ------------------------------ | ----------- |
| -32602 | Invalid block parameter        | Указан некорректный параметр блока |
| 4100   | Requested method not supported | Метод не поддерживается кошельком |

</Info>
