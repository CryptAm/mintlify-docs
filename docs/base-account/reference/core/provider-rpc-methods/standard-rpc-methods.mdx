Base Account поддерживает все стандартные Ethereum RPC методы, обеспечивая совместимость с существующими web3 приложениями и библиотеками. Каждый метод ниже содержит ссылку на подробную документацию, включая параметры, возвращаемые значения и обработку ошибок.

## Методы аккаунтов

### [eth_accounts](/base-account/reference/core/provider-rpc-methods/eth_accounts)

Возвращает список адресов, принадлежащих клиенту.

### [eth_requestAccounts](/base-account/reference/core/provider-rpc-methods/eth_requestAccounts)

Запрашивает у пользователя предоставление адреса Ethereum для идентификации. Этот метод используется для инициализации подключения между вашим приложением и кошельком пользователя.

## Обработка ошибок

Все RPC методы могут вызывать ошибки. Распространенные коды ошибок включают:

- `-32700`: Parse error (Ошибка парсинга)
- `-32600`: Invalid request (Некорректный запрос)
- `-32601`: Method not found (Метод не найден)
- `-32602`: Invalid params (Некорректные параметры)
- `-32603`: Internal error (Внутренняя ошибка)
- `-32000`: Server error (Ошибка сервера)

```javascript
try {
  const result = await provider.request({
    method: 'eth_getBalance',
    params: ['0x1234...', 'latest']
  });
} catch (error) {
  console.error('RPC Error:', error.code, error.message);
}
```

## Рекомендации

1. **Всегда обрабатывайте ошибки**: RPC вызовы могут завершиться неудачей по разным причинам
2. **Используйте соответствующие параметры блоков**: `latest`, `earliest`, `pending` или конкретные номера блоков
3. **Проверяйте адреса**: Убедитесь, что адреса правильно отформатированы
4. **Кэшируйте ответы**: Некоторые данные меняются нечасто
5. **Используйте пакетные запросы**: При поддержке выполняйте несколько вызовов пакетно для эффективности

## Совместимость

Base Account обеспечивает полную совместимость с:
- Web3.js
- Ethers.js
- Wagmi
- Viem
- Другими стандартными web3 библиотеками

Все стандартные RPC методы работают точно так, как ожидается, что позволяет вашим существующим приложениям интегрироваться с Base Account без модификаций.
