---
title: "eth_getTransactionByBlockNumberAndIndex"
description: "Получить транзакцию по номеру блока и позиции индекса транзакции"
---

Определено в [Спецификации Ethereum JSON-RPC](https://ethereum.org/en/developers/docs/apis/json-rpc/)

<Info>
Возвращает информацию о транзакции по номеру блока и позиции индекса транзакции.
</Info>

## Параметры

<ParamField body="blockParameter" type="string" required>
Номер блока в виде целого числа или строки "latest", "earliest" или "pending".
</ParamField>

<ParamField body="transactionIndex" type="string" required>
Позиция индекса транзакции в виде целого числа.
</ParamField>

## Возвращает

<ResponseField name="result" type="object">
Объект транзакции или `null`, если транзакция не найдена.

<Expandable title="Свойства объекта транзакции">
<ResponseField name="blockHash" type="string">
Хэш блока, в котором находилась эта транзакция.
</ResponseField>

<ResponseField name="blockNumber" type="string">
Номер блока, в котором находилась эта транзакция.
</ResponseField>

<ResponseField name="from" type="string">
Адрес отправителя.
</ResponseField>

<ResponseField name="gas" type="string">
Газ, предоставленный отправителем.
</ResponseField>

<ResponseField name="gasPrice" type="string">
Цена газа, предоставленная отправителем в Wei.
</ResponseField>

<ResponseField name="hash" type="string">
Хэш транзакции.
</ResponseField>

<ResponseField name="input" type="string">
Данные, отправленные вместе с транзакцией.
</ResponseField>

<ResponseField name="nonce" type="string">
Количество транзакций, выполненных отправителем до этой.
</ResponseField>

<ResponseField name="to" type="string">
Адрес получателя. `null` для транзакции создания контракта.
</ResponseField>

<ResponseField name="transactionIndex" type="string">
Позиция индекса транзакции в блоке.
</ResponseField>

<ResponseField name="value" type="string">
Переданное значение в Wei.
</ResponseField>

<ResponseField name="v" type="string">
Идентификатор (ID) восстановления ECDSA.
</ResponseField>

<ResponseField name="r" type="string">
Подпись ECDSA r.
</ResponseField>

<ResponseField name="s" type="string">
Подпись ECDSA s.
</ResponseField>
</Expandable>
</ResponseField>

<RequestExample>
```json Latest Block
{
  "id": 1,
  "jsonrpc": "2.0",
  "method": "eth_getTransactionByBlockNumberAndIndex",
  "params": [
    "latest",
    "0x0"
  ]
}
```

```json Specific Block
    "0x1b4",

  ]
}
```
</RequestExample>

## Обработка ошибок

| Код    | Сообщение                      | Описание    |
| ------ | ------------------------------ | ----------- |
| -32602 | Invalid parameters             | Некорректный параметр блока или индекс транзакции |
| 4100   | Requested method not supported | Метод не поддерживается кошельком |

<Info>
Индексы транзакций начинаются с 0x0 для первой транзакции в блоке. Если индекс превышает количество транзакций, возвращается `null`
</Info>
