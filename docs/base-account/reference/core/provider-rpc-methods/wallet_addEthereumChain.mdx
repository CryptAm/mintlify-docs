---
title: "wallet_addEthereumChain"
description: "Добавить новую цепь Ethereum в кошелек"
---

Определено в [EIP-3085](https://eips.ethereum.org/EIPS/eip-3085)

<Info>
Запрашивает у кошелька отслеживание указанной цепи и добавление её в пользовательский интерфейс кошелька. Этот метод используется для предложения пользователю новых сетей.
</Info>

## Параметры

<ParamField body="chainData" type="object" required>
Объект данных цепи, содержащий информацию о сети.

<Expandable title="Свойства данных цепи">
<ParamField body="chainId" type="string" required>
Идентификатор цепи (ChainID) в виде строки в шестнадцатеричном (hex) формате с префиксом "0x".
</ParamField>

<ParamField body="chainName" type="string" required>
Человекочитаемое название цепи.
</ParamField>

<ParamField body="nativeCurrency" type="object" required>
Объект, описывающий валюту, которая является "нативной" для этой цепи.

<Expandable title="Свойства нативной валюты">
<ParamField body="name" type="string" required>
Название нативной валюты.
</ParamField>

<ParamField body="symbol" type="string" required>
Символ нативной валюты.
</ParamField>

<ParamField body="decimals" type="number" required>
Количество десятичных знаков для нативной валюты.
</ParamField>
</Expandable>
</ParamField>

<ParamField body="rpcUrls" type="array" required>
Массив строк RPC URL.
</ParamField>

<ParamField body="blockExplorerUrls" type="array">
Массив строк URL обозревателя блоков (опционально).
</ParamField>

<ParamField body="iconUrls" type="array">
Массив строк URL иконок (опционально).
</ParamField>
</Expandable>
</ParamField>

## Возвращает

<ResponseField name="result" type="null">
Возвращает `null` в случае успеха.
</ResponseField>

<RequestExample>
```json Request
{
  "id": 1,
  "jsonrpc": "2.0",
  "method": "wallet_addEthereumChain",
  "params": [{
    "chainId": "0x2105",
    "chainName": "Base Mainnet",
    "nativeCurrency": {
      "name": "Ether",
      "symbol": "ETH",
      "decimals": 18
    },
    "rpcUrls": ["https://mainnet.base.org"],
    "blockExplorerUrls": ["https://basescan.org"]
  }]
}
```
</RequestExample>

<ResponseExample>
```json Success Response
{
  "id": 1,
  "jsonrpc": "2.0",
  "result": null
}
```
</ResponseExample>

## Обработка ошибок

| Код  | Сообщение                      | Описание    |
| ---- | ------------------------------ | ----------- |
| 4001 | User rejected the request      | Пользователь отклонил добавление цепи |
| 4100 | Requested method not supported | Метод не поддерживается кошельком |
| 4902 | Unrecognized chain ID          | Идентификатор цепи (chainID) не распознан |

<Warning>
Убедитесь, что вы предоставляете точные RPC URL и URL обозревателей блоков. Некорректные URL могут вызвать проблемы с подключением.
</Warning>
