---
title: "web3_clientVersion"
description: "Получить текущую версию клиента"
---

Определено в [Спецификации Ethereum JSON-RPC](https://ethereum.org/en/developers/docs/apis/json-rpc/)

<Info>
Возвращает текущую версию клиента.
</Info>

## Параметры

Этот метод не принимает параметров.

## Возвращает

<ResponseField name="result" type="string">
Строка с текущей версией клиента.
</ResponseField>

<RequestExample>
```json Request
{
  "id": 1,
  "jsonrpc": "2.0",
  "method": "web3_clientVersion",
  "params": []
}
```
</RequestExample>

<ResponseExample>
```json Success Response
{
  "id": 1,
  "jsonrpc": "2.0",
  "result": "CoinbaseWallet/4.0.0"
}
```
</ResponseExample>
