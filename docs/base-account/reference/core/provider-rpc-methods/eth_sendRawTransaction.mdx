---
title: "eth_sendRawTransaction"
description: "Отправить подписанную транзакцию в сеть"
---

Определено в [Спецификации Ethereum JSON-RPC](https://ethereum.org/en/developers/docs/apis/json-rpc/)

<Info>
Отправляет предварительно подписанную транзакцию для распространения в сети Ethereum..
</Info>

## Параметры

<ParamField body="signedTransactionData" type="string" required>
Данные подписанной транзакции в виде шестнадцатеричной строки (hex).
</ParamField>

## Возвращает

<ResponseField name="result" type="string">
Хэш транзакции (32 байта) в виде шестнадцатеричной строки (hex) или нулевой хэш, если транзакция ещё недоступна.
</ResponseField>

<RequestExample>
```json Request
{
  "id": 1,
  "jsonrpc": "2.0",
  "method": "eth_sendRawTransaction",
  "params": ["0xd46e8dd67c5d32be8d24c6b0afe7c5c3f4e9c3b2dae18d0c6b0cf5c8f3e8b2c1"]
}
```
</RequestExample>

<ResponseExample>
```json Success Response
{
  "id": 1,
  "jsonrpc": "2.0",
  "result": "0xe670ec64341771606e55d6b4ca35a1a6b75ee3d5145a99d05921026d1527331"
}
```
</ResponseExample>

## Обработка ошибок

| Код    | Сообщение                      | Описание    |
| ------ | ------------------------------ | ----------- |
| -32602 | Invalid params                 | Некорректный формат данных транзакции |
| -32000 | Insufficient funds             | Недостаточно средств на аккаунте |
| 4001   | User rejected the request      | Пользователь отклонил запрос на транзакцию |
| 4100   | Requested method not supported | Метод не поддерживается кошельком |

<Warning>
Убедитесь, что транзакция правильно подписана перед отправкой. Некорректные подписи приведут к неудаче транзакции.
</Warning>
