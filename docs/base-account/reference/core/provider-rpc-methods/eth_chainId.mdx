---
title: "eth_chainId"
description: "Получение идентификатора (ID) текущей настройки цепочки"
---

Определено в [EIP-695](https://github.com/ethereum/EIPs/blob/master/EIPS/eip-695.md)

<Info>
Возвращает идентификатор текущей настройки цепочки (chain ID), значение, используемое при подписании защищенных от повторного использования транзакций, введенных EIP-155.
</Info>

## Параметры

Этот метод не принимает параметров.

## Возвращает

<ResponseField name="result" type="string">
Строка в шестнадцатеричном формате, представляющая идентификатор (ID) цепочки.
</ResponseField>

<RequestExample>
```json Request
{
  "id": 1,
  "jsonrpc": "2.0",
  "method": "eth_chainId",
  "params": []
}
```
</RequestExample>

<ResponseExample>
```json Ethereum Mainnet
{
  "id": 1,
  "jsonrpc": "2.0",
  "result": "0x1"
}
```

```json Base Mainnet
{
  "id": 1,
  "jsonrpc": "2.0",
  "result": "0x2105"
}
```
</ResponseExample>

## Обработка ошибок

| Код  | Сообщение                      | Описание    |
| ---- | ------------------------------ | ----------- |
| 4100 | Requested method not supported | Метод не поддерживается кошельком |

<Info>
Общие идентификаторы цепочек: Ethereum Mainnet (0x1), Base Mainnet (0x2105), Base Sepolia (0x14a34).
</Info>
