---
title: "eth_getBlockByHash"
description: "Получить информацию о блоке по хэшу блока"
---

Определено в [Спецификации Ethereum JSON-RPC](https://ethereum.org/en/developers/docs/apis/json-rpc/)

<Info>
Возвращает информацию о блоке по его хэшу.
</Info>

## Параметры

<ParamField body="blockHash" type="string" required>
Хэш блока (32 байта).
</ParamField>

<ParamField body="fullTransactionObjects" type="boolean" required>
Если `true`, возвращает полные объекты транзакций; если `false`, возвращает только хэши транзакций.
</ParamField>

## Возвращает

<ResponseField name="result" type="object">
Объект блока или `null`, если блок не найден.

<Expandable title="Свойства объекта блока">
<ResponseField name="number" type="string">
Номер блока (hex). `null` для ожидающего блока.
</ResponseField>

<ResponseField name="hash" type="string">
Хэш блока (hex). `null` для ожидающего блока.
</ResponseField>

<ResponseField name="parentHash" type="string">
Хэш родительского блока (hex).
</ResponseField>

<ResponseField name="nonce" type="string">
Хэш доказательства выполнения работы (hex). `null` для ожидающего блока.
</ResponseField>

<ResponseField name="sha3Uncles" type="string">
SHA3 хэш данных дядей в блоке (hex).
</ResponseField>

<ResponseField name="logsBloom" type="string">
Bloom-фильтр для логов блока (hex). `null` для ожидающего блока.
</ResponseField>

<ResponseField name="transactionsRoot" type="string">
Корень дерева транзакций блока (hex).
</ResponseField>

<ResponseField name="stateRoot" type="string">
Корень дерева финального состояния блока (hex).
</ResponseField>

<ResponseField name="receiptsRoot" type="string">
Корень дерева квитанций блока (hex).
</ResponseField>

<ResponseField name="miner" type="string">
Адрес получателя вознаграждения за майнинг.
</ResponseField>

<ResponseField name="difficulty" type="string">
Сложность этого блока в виде целого числа (hex).
</ResponseField>

<ResponseField name="totalDifficulty" type="string">
Совокупная сложность цепочки до этого блока в виде целого числа (hex).
</ResponseField>

<ResponseField name="extraData" type="string">
Поле "дополнительных данных" этого блока (hex).
</ResponseField>

<ResponseField name="size" type="string">
Размер этого блока в байтах в виде целого числа (hex).
</ResponseField>

<ResponseField name="gasLimit" type="string">
Максимально допустимый газ в этом блоке (hex).
</ResponseField>

<ResponseField name="gasUsed" type="string">
Общий газ, использованный всеми транзакциями в этом блоке (hex).
</ResponseField>

<ResponseField name="timestamp" type="string">
Unix-метка времени создания блока (hex).
</ResponseField>

<ResponseField name="transactions" type="array">
Массив объектов транзакций или 32-байтных хэшей транзакций в зависимости от параметра `fullTransactionObjects`.
</ResponseField>

<ResponseField name="uncles" type="array">
Массив хешей uncle-блоков.
</ResponseField>
</Expandable>
</ResponseField>

## Обработка ошибок

| Код    | Сообщение                      | Описание    |
| ------ | ------------------------------ | ----------- |
| -32602 | Invalid block hash             | Указан некорректный хэш блока |
| 4100   | Requested method not supported | Метод не поддерживается кошельком |

<Info>
Установка `fullTransactionObjects` в `true` возвращает полные детали транзакций, а `false` возвращает только хэши транзакций для лучшей производительности.
</Info>
