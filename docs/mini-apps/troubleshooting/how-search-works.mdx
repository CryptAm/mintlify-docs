---
Индексация - это процесс, с помощью которого приложение Base добавляет ваше мини-приложение в свой каталог, делая его доступным для обнаружения через поиск и просмотр.
В отличие от традиционных магазинов приложений с ручной отправкой, вы контролируете, когда происходит индексация. Поделитесь URL вашего мини-приложения в социальной ленте, и индексация начнется автоматически - никакого процесса проверки не требуется.

## Как работает индексация

Понимание процесса индексации поможет вам диагностировать, почему ваше мини-приложение может не появляться.

<Note>
Ваш манифест должен быть правильно настроен и проверен, чтобы индексация работала. См. [документацию манифеста](/mini-apps/core-concepts/manifest) для информации об обязательных полях и валидации.
</Note>

<Steps>
<Step title="Поделитесь URL вашего мини-приложения">
  Поделитесь вашим URL в ленте.
</Step>
<Step title="Приложение Base проверяет ваш манифест">
  Приложение Base загружает и проверяет ваш файл манифеста.
  <Warning>
  Некорректные или недоступные манифесты приведут к сбою индексации.
  </Warning>
</Step>
<Step title="Ваше приложение попадает в каталог">
  Ваше мини-приложение регистрируется и становится доступным для поиска в течение 10 минут.
</Step>
</Steps>


## Как работает поиск
Поиск в приложении Base запрашивает поле name вашего мини-приложения из каталога. В результатах появляются как точные, так и частичные совпадения. Поиск отображает ваше мини-приложение на основе информации, которую вы предоставили в манифесте.

<img src="/images/minikit/search.jpg" alt="search bar in the Base app" className="h-18 w-auto" />
</Frame>

<Note>
При внесении изменений в ваш манифест вам потребуется поделиться URL, чтобы переиндексировать его.
</Note>

## Площадки для обнаружения

### Просмотр по категориям
Ваше приложение появляется в категории, указанной в поле `primaryCategory` вашего манифеста. Пользователи просматривают категории, чтобы находить приложения по интересам.

<Frame caption="Категории приложений">
<img src="/images/minikit/categories.jpg" alt="app categories" className="h-18 w-auto"/>
</Frame>

<Note>
**Ранжирование категорий**: Ранжирование использует метрики вовлеченности за 7 дней, такие как расшаривания.
</Note>

### Сохраненные приложения
Когда пользователи сохраняют ваше мини-приложение, оно появляется в их личном списке сохраненных опций для быстрого доступа. Предлагайте пользователям сохранять приложение в ключевые моменты.

<Frame caption="Сохраненные приложения">
<img src="/images/minikit/my-apps.jpg" alt="saved apps" className="h-18 w-auto"/>
</Frame>

### Личные сообщения

Когда пользователи делятся URL вашего мини-приложения в личном сообщении, оно отображается как интерактивное встраивание. Получатели могут просмотреть предварительный просмотр и открыть ваше приложение прямо из беседы, при этом `context.location` будет установлен в значение `messaging`, что позволит вам настроить опыт для совместного обнаружения.
