Приложение Base работает над полной совместимостью с Farcaster Mini App SDK. Во время бета-тестирования некоторые функции еще не поддерживаются.

## В настоящее время не поддерживается
- [`signManifest` (экспериментальная)](https://miniapps.farcaster.xyz/docs/sdk/actions/sign-manifest)

## Определение клиента Base App
Чтобы определить, запущено ли приложение в Base App, вы можете использовать свойство `clientFid` объекта `context`.

```tsx App.tsx
import { useMiniKit } from '@coinbase/onchainkit/minikit';

function MyComponent() {
  const { context } = useMiniKit();
  const isBaseApp = context.client.clientFid === 309857;

  if (isBaseApp) {
    // Используйте функции, специфичные для Base App
    console.log('Running in Base App');
  }
  
  return <div>{/* Your component */}</div>;
}
```
