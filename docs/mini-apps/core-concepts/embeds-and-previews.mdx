## Реализация
Добавьте этот метатег в секцию `<head>` любой страницы, которой вы хотите поделиться:

<CodeGroup>
  ```html html metadata
  <!DOCTYPE html>
  <html lang="en">
    <head>
      <title>My Mini App</title>
      <meta
        name="fc:miniapp"
        content='{
          "version": "next",
          "imageUrl": "https://example.com/preview.png",
          "button": {
            "title": "Open App",
            "action": {
              "type": "launch_frame",
              "url": "https://example.com"
            }
          }
        }'
      />
    </head>
    <body>
      <!-- Your app content -->
    </body>
  </html>
  ```
  
  ```jsx next.js metadata
  // app/layout.tsx or app/page.tsx (Next.js App Router)
  import type { Metadata } from "next";
  
export async function generateMetadata(): Promise<Metadata> {
  return {
    title: miniapp.name,
    description: miniapp.description,
    other: {
      "fc:miniapp": JSON.stringify({
        version: miniapp.version,
        imageUrl: miniapp.heroImageUrl,
        button: {
          title: `Join the ${miniapp.name}`,
          action: {
            name: `Launch ${miniapp.name}`,
            url: `${miniapp.homeUrl}`
          },
        },
      }),
    },
  };
}
  
  export default function RootLayout({ children }: { children: React.ReactNode }) {
    return (
      <html lang="en">
        <body>{children}</body>
      </html>
    );
  }
  ```
</CodeGroup>
<Tip>The `homeUrl`, используемый в `манифесте`, *должен* иметь определённые метаданные для встраивания, чтобы мини-приложение корректно отображалось в Base app.</Tip>



## Схема

<Card>
<ParamField path="version" type="string" required>
Версия встраивания. Должна быть `"1"` или `"next"`.
</ParamField>

<ParamField path="imageUrl" type="string" required>
URL изображения для встраивания. Должен быть соотношением сторон 3:2, максимум 10MB, максимум 1024 символа.
</ParamField>

<ParamField path="button" type="object" required>
Объект конфигурации кнопки.
</ParamField>
</Card>

### Конфигурация кнопки

Определяет кнопку запуска, которая появляется на встраивании.

<Card>
<ParamField path="button.title" type="string" required>
Текст кнопки. Максимум 32 символа.
</ParamField>

<ParamField path="button.action" type="object" required>
Объект конфигурации действия. Максимум 1024 символа.
</ParamField>
</Card>

### Конфигурация действия

Определяет, что происходит при нажатии кнопки встраивания.

<Card>
<ParamField path="button.action.type" type="string" required>
Тип действия. Должен быть `"launch_frame"`.
</ParamField>

<ParamField path="button.action.url" type="string">
URL приложения для открытия. По умолчанию полный URL страницы, включая параметры запроса. Максимум 1024 символа.
</ParamField>

<ParamField path="button.action.name" type="string">
Название приложения. Максимум 32 символа. По умолчанию используется название из манифеста.
</ParamField>

<ParamField path="button.action.splashImageUrl" type="string">
URL изображения экрана-заставки. Должно быть 200x200 пикселей. Максимум 32 символа. По умолчанию используется изображение заставки из манифеста.
</ParamField>

<ParamField path="button.action.splashBackgroundColor" type="string">
Цвет фона экрана-заставки. Должен быть hex-кодом цвета. По умолчанию используется цвет фона заставки из манифеста.
</ParamField>
</Card>
