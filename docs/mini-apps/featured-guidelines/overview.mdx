---

Ваше приложение должно соответствовать всем продуктовым, дизайнерским и техническим требованиям, изложенным ниже. Соответствие этим требованиям является обязательным условием для попадания в рекомендованные, но **не гарантирует размещения**. Base предъявляет очень высокую планку для попадания в рекомендованные.


<Note>
  Чтобы подать заявку на размещение вашего приложения в рекомендованных, сначала подтвердите свое мини-приложение в [панели управления Base Build](https://base.dev/), затем заполните [форму отправки](https://buildonbase.deform.cc/getstarted/).
</Note>


<Steps titleSize="h3">
<Step title="Аутентификация">
  * Аутентификация внутри приложения остается в пределах приложения Base без внешних перенаправлений
  * Подключение кошелька происходит автоматически
  * Нет подтверждения по электронной почте или телефону внутри приложения
</Step>

<Step title="Онбординг">
  * Объясните цель приложения и как начать работу, с четкими инструкциями по онбордингу либо на главной странице, либо во всплывающем окне.
  * Приложение запрашивает только необходимую личную информацию, с ясным контекстом
  * Отображается аватар и имя пользователя **(без 0x адресов)**
</Step>

<Step title="Совместимость с Base">
  * Приложение является клиент-агностиком, без жестко закодированного текста или ссылок, специфичных для Farcaster, или другого поведения, специфичного для конкретного клиента
  * Транзакции спонсируются
</Step>

<Step title="Макет">
  * Призывы к действию (CTA) видны и расположены по центру страницы
  * У приложения есть нижняя панель навигации или боковое меню для легкого доступа к основному потоку
  * Все кнопки доступны и не обрезаны
  * Элементы панели навигации имеют четкие, понятные подписи
</Step>

<Step title="Время загрузки">
  * Приложение загружается в течение **3 секунд**

  * Действия внутри приложения выполняются в течение **1 секунды**
  
  * Во время действий отображаются индикаторы загрузки
</Step>

<Step title="Удобство использования">
  * Приложение последовательно поддерживает **светлую и темную темы**
  * У приложения минимальные цели для касания **44px**
</Step>

<Step title="Метаданные приложения">
  * Описание приложения четкое, лаконичное и ориентировано на пользователя
  * Иконка приложения имеет размер **1024×1024 px**, формат PNG, **без прозрачности**, читаема в маленьких размерах
  * Обложка приложения **высокого качества** и не содержит логотипа Base или фотографий команды
</Step>
</Steps>
