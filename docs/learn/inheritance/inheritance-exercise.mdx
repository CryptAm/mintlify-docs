Создайте контракты, соответствующие следующим спецификациям.

---

## Контракты

### Employee

Создайте `abstract` контракт под названием `Employee`. Он должен иметь:

- Публичную переменную, хранящую `idNumber`
- Публичную переменную, хранящую `managerId`
- Конструктор, который принимает аргументы для обеих этих переменных и устанавливает их
- Виртуальную функцию `getAnnualCost`, которая возвращает `uint`

### Salaried

Контракт под названием `Salaried`. Он должен:

- Наследовать от `Employee`
- Иметь публичную переменную для `annualSalary`
- Реализовать `override` функцию для `getAnnualCost`, которая возвращает `annualSalary`
- Соответствующий конструктор, который выполняет любую настройку, включая установку `annualSalary`

### Hourly

Реализуйте контракт под названием `Hourly`. Он должен:

- Наследовать от `Employee`
- Иметь публичную переменную, хранящую `hourlyRate`
- Включать любую другую необходимую настройку и реализацию

<Tip>
Годовая стоимость почасового сотрудника - это его почасовая ставка * 2080 часов.
</Tip>


### Manager

Реализуйте контракт под названием `Manager`. Он должен:

- Иметь публичный массив, хранящий идентификаторы сотрудников (employee Ids)
- Включать функцию `addReport`, которая может добавлять номера идентификаторов в этот массив
- Включать функцию `resetReports`, которая может сбрасывать этот массив до пустого состояния

### Salesperson

Реализуйте контракт под названием `Salesperson`, который наследует от `Hourly`.

### Engineering Manager

Реализуйте контракт под названием `EngineeringManager`, который наследует от `Salaried` и `Manager`.

## Развертывания

Для этого упражнения вам нужно будет выполнить более сложный набор развертываний.

Разверните свои контракты `Salesperson` и `EngineeringManager`. Вам не нужно отдельно развертывать другие контракты.

Используйте следующие значения:

### Salesperson

- Почасовая ставка составляет 20 долларов в час
- Номер идентификатора (ID) - 55555
- Номер идентификатора менеджера 12345

### Manager

- Годовая зарплата - 200,000
- Номер идентификатора (ID) - 54321
- Номер идентификатора менеджера (ID) - 11111

## Отправка наследования

Скопируйте приведенный ниже контракт и разверните его, используя адреса ваших контрактов `Salesperson` и `EngineeringManager`.

```solidity
contract InheritanceSubmission {
    address public salesPerson;
    address public engineeringManager;

    constructor(address _salesPerson, address _engineeringManager) {
        salesPerson = _salesPerson;
        engineeringManager = _engineeringManager;
    }
}
```
