# Коннекторы кошельков

Одна из самых пугающих задач при создании ончейн-приложения — установить первоначальное соединение между кошельками ваших пользователей и вашим приложением. Первоначальное исследование часто выявляет ошеломляющее количество кошельков, каждый со своими собственными SDK и собственными методами управления подключением. К счастью, вам не нужно управлять всем этим самостоятельно. Существует ряд библиотек-коннекторов кошельков, специализирующихся на создании плавного и красивого пользовательского опыта для облегчения этого подключения.

Чтобы еще больше добавить путаницы и сложности, [Smart wallets] набирают популярность. Эти продвинутые кошельки позволяют пользователям создавать и управлять кошельками с помощью [passkeys],и поддерживают (или скоро будут поддерживать) растущий набор функций, включая сессионные ключи, восстановление аккаунта и многое другое!

[RainbowKit], агрегатор, который вы будете использовать в этом уроке, работает с Coinbase Smart Wallet "из коробки", но вам потребуется немного дополнительной настройки, чтобы поддерживать пользователей как традиционных кошельков, так и смарт-кошельков.

## Подключение к блокчейну

Одна из многих сложных задач создания фронтенда, который может взаимодействовать с вашими смарт-контрактами, - управление подключением пользователя между вашим ончейн-приложением и его кошельком [EOA]. Мало того, что существует постоянно растущий набор различных кошельков, пользователи могут (и, вероятно, должны!) использовать несколько разных адресов в одном и том же кошельке.

[Rainbowkit] - один из нескольких вариантов, который немного упрощает эту задачу, выступая в роли агрегатора кошельков и обрабатывая некоторые детали их подключения. Альтернативы включают [ConnectKit], и [Dynamic], которые также являются отличными вариантами.

Каждый из них включает настраиваемые UI/UX компоненты для приглашения пользователя подключиться, отображения статуса подключения и выбора кошелька, который он хочет использовать.

### Использование быстрого старта

Если вы просто пытаетесь запустить проект как можно быстрее, вы можете использовать скрипт RainbowKit [quick start] для создания приложения из их шаблона одной командой. Если вы используете Yarn:

```bash
yarn create @rainbow-me/rainbowkit
```

<Note>
Скрипт не принимает `.` в качестве имени проекта, поэтому вам нужно будет запустить этот скрипт в вашей директории `src` или там, где вы храните свои проекты. Он создаст папку с тем же именем, что и ваш проект, и установит файлы проекта внутри.
</Note>


После завершения просто запустите приложение:

```bash
yarn run dev
```

Использование скрипта быстро, но это означает меньше выбора. В данном случае он строит приложение на основе [Next.js], что отлично, если вы хотите его использовать, но не помогает, если вы предпочитаете работать с другим фреймворком, таким как [Create React App], или [Remix] (the React framework, not the Solidity IDE). Скрипт также не поможет вам, если вы хотите добавить ончейн-интеграцию на существующий сайт.

<Note>
Шаблон RainbowKit был обновлен до wagmi 2.X, но он **не** использует App Router от Next.js. Вам нужно будет установить его вручную, если вы хотите использовать новейшие паттерны.

Руководство [Building an Onchain App] покажет вам, как это сделать!
</Note>


### Coinbase Smart Wallet

Если у вас есть расширение Coinbase Wallet, вам может быть интересно, где можно найти смарт-кошелек. По умолчанию смарт-кошелек будет вызываться только если вы нажмете кнопку `Coinbase Wallet` для входа **и** у вас **нет** браузерного расширения. Чтобы протестировать, откройте приватное окно с отключенными расширениями и попробуйте войти.

Выбор `Rainbow`, `MetaMask` или `WalletConnect` покажет QR-код, чтобы пользователь мог войти со своего телефона. Выбор `Coinbase Wallet` вместо этого вызовет вход через смарт-кошелек.

Этот поток можно улучшить, так как новые пользователи криптовалют не будут знать, что поиск смарт-кошелька - лучший путь вперед, а существующие пользователи, пытающиеся перейти на смарт-кошелек, не имеют этой опции.

Смотрите наше руководство о том [Use the Coinbase Smart Wallet and EOAs with OnchainKit] как для более подробной информации!
